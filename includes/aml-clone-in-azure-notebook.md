### <a name="azure-notebooks---free-jupyter-based-notebooks-in-the-azure-cloud"></a>**Azure Notebooks** - Azure クラウドでの無料の Jupyter ベースのノートブック
Azure Notebooks には SDK が既にインストールされて構成されています。
  
1. ワークスペースを作成して Azure Notebooks を開始するには、[使用開始クイック スタート](../articles/machine-learning/service/quickstart-get-started.md)を行ってください。
1. [Azure Notebooks](https://notebooks.azure.com/) に移動する
1. クイック スタートで作成した `Getting Started` ライブラリで、`tutorials` フォルダーに移動します
1. ノートブックを開きます。

### <a name="your-own-jupyter-notebook-server"></a>**自分の Jupyter Notebook サーバー**
1. SDK をインストールしてワークスペースを作成するには、[Python SDK での作業開始クイック スタート](../articles/machine-learning/service/quickstart-create-workspace-with-python.md)を行います。
1. [GitHub リポジトリ](https://aka.ms/aml-notebooks)を複製します。
1. クイック スタートの間に作成した `aml_config` ディレクトリを、複製したディレクトリにコピーします。
1. 複製したディレクトリから、Notebook サーバーを起動します。
1. `tutorials` フォルダーに移動します。
1. ノートブックを開きます。
