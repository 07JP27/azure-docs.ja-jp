### 最新の変更点

###### リファレンスの追加

Azure Storage NuGet パッケージが Visual Studio プロジェクトに追加されました。
このパッケージによって次の .NET リファレンスが追加されます。Microsoft.Data.Edm、Microsoft.Data.OData、Microsoft.Data.Services.Client、Microsoft.WindowsAzure.Configuration、Microsoft.WindowsAzure.Storage、Newtonsoft.Json、System.Data、System.Spatial

###### Azure Storage の接続文字列の追加

選択されたストレージ アカウントの接続文字列とキーを使用して、プロジェクトの web.config ファイル内に要素が作成されました。
