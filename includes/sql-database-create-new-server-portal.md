## Azure SQL Database 論理サーバーを作成する

次の手順に従い、Azure ポータルで Azure SQL Database 論理サーバーを作成します。

1. [Azure ポータル](http://portal.azure.com)に接続していない場合は接続します。
2. **[新規]** をクリックし、「**SQL Database**」と入力し、**[SQL Database (新しい論理データベース)]** をクリックします。

      ![新しい論理サーバー](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-1.png)

3. [SQL Database (新しい論理データベース)] をクリックします。

      ![新しい論理サーバー](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-2.png)
   
4. **[作成]** をクリックしてテンプレートを開き、単一データベースとエラスティック データベース プールをホストできる空の論理サーバーを作成します。

      ![新しい論理サーバー](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-3.png)

5. 次のサーバー プロパティの値を入力します。

 - サーバー名
 - サーバー管理者のログイン
 - パスワード
 - サブスクリプション (複数のサブスクリプションがある場合のみ)
 - リソース グループ (新規または既存)
 - 場所

        ![new logical server](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-4.png)

6.  **[作成]** をクリックします。通知領域にデプロイの開始が表示されます。

       ![新しい論理サーバー](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-5.png)

7. デプロイが完了するまで待ってから、次の手順に進みます。

       ![新しい論理サーバー](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-6.png)
