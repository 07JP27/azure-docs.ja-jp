## <a name="sign-in-to-azure"></a>Azure へのサインイン

Azure CLI を使用して、Azure でアプリをホストするために必要なリソースを作成します。 Cloud Shell で Azure CLI コマンドを実行する場合は、既にサインインしています。 Azure CLI コマンドをローカルで実行するには、[az login](/cli/azure/#login) コマンドで Azure サブスクリプションにサインインし、画面の指示に従います。

```azurecli
az login
```
