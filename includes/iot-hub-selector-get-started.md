## <a name="introduction"></a>概要
Azure IoT Hub は、完全に管理されたサービスを信頼性の高い有効にして、何百万ものモ ノのインターネット (IoT) デバイスと、ソリューションの間で双方向通信をセキュリティで保護されたバック エンドです。 IoT が面をプロジェクトの最大の課題の 1 つは、信頼性と安全には、バックエンドのソリューションにデバイスを接続する方法です。 IoT Hub、課題に対処します。

* 信頼できるデバイスとクラウドとクラウドからデバイスするハイパー スケール メッセージングを提供します。
* デバイスごとのセキュリティ資格情報を使用して通信をセキュリティで保護され、アクセス制御を有効にします。
* 最も一般的な言語とプラットフォームのデバイスのライブラリが含まれます。

このチュートリアルではどのようにします。

* IoT hub を作成するのにには、Azure ポータルを使用します。
* IoT ハブにデバイス id を作成します。
* ソリューション、バック エンドにテレメトリを送信するシミュレートされたデバイスのアプリを作成します。
