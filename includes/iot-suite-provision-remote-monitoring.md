## <a name="provision-the-solution"></a>ソリューションをプロビジョニングします。

リモートの監視の構成済みソリューションは、自分のアカウントで既にプロビジョニングしていない: 場合

1. ログオン[azureiotsuite.com] [ lnk-azureiotsuite]アカウントの資格情報、およびをクリックして、Azure を使用して **+** ソリューションを作成します。
2. をクリックして**選択**上、**リモート監視**タイルです。
3. 入力、**ソリューション名**ソリューションが事前に構成されて、リモートの監視用です。
4. 選択、**地域**と**サブスクリプション**ソリューションのプロビジョニングに使用します。
5. をクリックして**ソリューションを作成する**プロビジョニング プロセスを開始します。 このプロセスは、実行するのに数分をとります。

### <a name="wait-for-the-provisioning-process-to-complete"></a>プロビジョニング プロセスを完了するまで待機します。
1. ソリューションのタイルをクリックして**プロビジョニング**状態です。
2. 通知、**プロビジョニング状態**Azure サブスクリプションで Azure サービスが配置されています。
3. プロビジョニングが完了すると、状態に変更されて**準備**です。
4. 右側のペインで、ソリューションの詳細を表示するタイルをクリックします。

> [!NOTE]
> 構成済みのソリューションの配置の問題が発生する場合は、確認[azureiotsuite.com サイトに対するアクセス許可][ lnk-permissions]と[FAQ][lnk-faq]です。 問題が解決しない場合に、サービス チケットを作成、[ポータル][lnk-portal]です。
> 
> 

詳細、ソリューションの一覧にないことを確認する必要がありますか。 機能に関する意見を与える[User Voice](https://feedback.azure.com/forums/321918-azure-iot)です。
