1. 仮想ネットワークの **[設定]** ページで、 **[DNS サーバー]** に移動し、クリックして **[DNS サーバー]** ページを開きます。

    ![DNS サーバーの指定](./media/vpn-gateway-specify-dns-portal-include/specify-dns-server.png "DNS サーバーの指定")

   - **[DNS サーバー]:** **[カスタム]** を選択します。
   - **[DNS サーバーの追加]:** 名前解決に使用する DNS サーバーの IP アドレスを入力します。

2. DNS サーバーを追加し終わったら、ページの上部の **[保存]** をクリックします。
