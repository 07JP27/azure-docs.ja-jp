

このサンプルを実行するには、最新の [Azure CLI 2.0](https://docs.microsoft.com/cli/azure/install-azure-cli) がインストールされていることを確認してください。 開始するには、`az login` を実行して、Azure との接続を作成します。

このサンプルは、bash シェルに対応しています。 Azure CLI スクリプトを Windows クライアントで実行する方法については、[Windows での Azure CLI の実行](../articles/virtual-machines/windows/cli-options.md)に関する記事を参照してください。
