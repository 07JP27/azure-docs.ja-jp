1. 管理者としてプロセス サーバーにサインインします。
2. [統合セットアップ](https://aka.ms/unifiedinstaller)の最新バージョンをダウンロードします。
3. インストーラーをダブルクリックして、更新プロセスを開始します。
4. インストーラーにより、インストールされているさまざまなコンポーネントが検出され、最新バージョンにアップグレードされます。
