* LUIS の**[オーサリング キー](../articles/cognitive-services/luis/luis-concept-keys.md#authoring-key)**。 
* LUIS-Samples の GitHub リポジトリから [TravelAgent アプリ](https://github.com/Microsoft/LUIS-Samples/blob/master/documentation-samples/quickstarts/change-model/TravelAgent.json)をインポートします。 
* インポートした TravelAgent アプリ用の LUIS [**アプリケーション ID**](../articles/cognitive-services/luis/luis-get-started-create-app.md)。 アプリケーション ID は、アプリケーション ダッシュボードに表示されます。 
* インポートする発話の例を含む **[utterances.json](https://github.com/Microsoft/LUIS-Samples/blob/master/documentation-samples/quickstarts/change-model/utterances.json)** ファイル。 
* 発話を受け取るアプリケーション内の**バージョン ID**。 既定の ID は "0.1" です。
