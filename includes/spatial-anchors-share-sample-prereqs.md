## <a name="prerequisites"></a>前提条件

このチュートリアルを完了するには、以下のものが必要です。

* **ASP.NET および Web の開発**ワークロードと共に <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017+</a> がインストールされている Windows マシン。
* [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download)。
* デプロイと実行用の 1 つ以上のデバイス (iOS または Android)。
  * Android を使用する場合:
    * Windows マシンにインストールされた <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3 以降</a>と <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3 以降</a>。
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">開発者向け</a>の <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore 対応</a> Android デバイス。
  * iOS を使用する場合:
    * <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10 以降</a>、<a href="https://cocoapods.org" target="_blank">CocoaPods</a>、<a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3 以降</a>がインストールされている macOS マシン。
    * 開発者向けの <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit 対応</a> iOS デバイス。
