1. Azure Portal にアクセスします。
2. Azure portal の **[リソースの作成]** ボタンを選択して、新しいリソースを追加します。
3. **[Serverless Function App]\(サーバーレス関数アプリ\)** を選択します
4. 作成設定にすべての情報を入力し、**[ホスティング プラン]** を **[App Service プラン]** に設定します。
    
    ![関数アプリ プランの作成](./media/functions-premium-create/create-function-app-plan.png)  

    1. サービス プランに対して Premium プラン (EP) レベルを選択します。 
        ![Premium プランの選択](./media/functions-premium-create/hosting-plan.png)  

    1. 完了したプランの設定を検証します。  
        ![完成した App Service プラン](./media/functions-premium-create/create-app-service-plan.png)  
    
1. アプリとプランの設定を検証し、**[作成]** を選択します。

    ![完成した App Service プラン](./media/functions-premium-create/create-function-app.png)
