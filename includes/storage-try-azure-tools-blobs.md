> [!TIP]
> 
> Azure Storage Explorer を使用して Azure Blob Storage リソースを管理します。 
> [Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/) は、[Azure Blob Storage のリソースを管理](../articles/vs-azure-tools-storage-explorer-blobs.md)できる Microsoft 製の無料のスタンドアロン アプリです。 Azure Storage Explorer を使用して、BLOB コンテナーおよび BLOB を視覚的に作成、読み取り、更新、削除できるほか、BLOB コンテナーおよび BLOB へのアクセスを管理できます。
