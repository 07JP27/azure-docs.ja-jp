### <a name="azure-active-directory-authentication"></a>Azure Active Directory 認証

Azure Spatial Anchors により、アプリケーションは Azure Active Directory のユーザー トークンでも認証できるようになります。 たとえば、AAD トークンを使用して ASA と統合することができます。 企業が AAD でユーザーを管理している場合は、Azure Spatial Anchors SDK でユーザー AAD トークンを提供できます。 こうすることにより、同じ AAD テナントの一部であるアカウントの ASA サービスに直接認証することができます。
