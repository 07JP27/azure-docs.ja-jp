Cognitive Services または Azure を初めて使用する場合は、[一時キー](https://azure.microsoft.com/try/cognitive-services/)を作成します。 Azure アカウントがある場合は、ポータルで新しいリソースを作成します。 Language Understanding エンドポイントのクエリでは、2 種類の Azure リソース (**Language Understanding** または **Cognitive Service** リソース) がサポートされます。 キーをアプリに[割り当てる](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md)ときに必要です。
