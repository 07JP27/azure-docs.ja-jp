**[Manage]\(管理\)** セクション (右上のメニュー) の **[Keys and endpoints]\(キーとエンドポイント\)** ページ (左側のメニュー) で、ページの下部にある**エンドポイント** URL を選択します。 別のブラウザー タブが開き、アドレス バーにエンドポイント URL が表示されます。 

エンドポイント URL は `https://<region>.api.cognitive.microsoft.com/luis/v2.0/apps/<appID>?verbose=true&subscription-key=<YOUR_KEY>&<optional-name-value-pairs>&q=<user-utterance-text>` のようになります。
