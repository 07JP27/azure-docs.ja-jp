#### <a name="to-add-a-storage-account-credential-in-the-same-azure-subscription-as-the-storsimple-device-manager-service"></a>StorSimple デバイス マネージャーのサービスと同じ Azure サブスクリプションのストレージ アカウントの資格情報を追加するには

1. StorSimple デバイス Manager サービスに移動します。 **構成**セクションで、**ストレージ アカウントの資格情報**です。

    ![ストレージ アカウントの資格情報](./media/storsimple-8000-configure-new-storage-account-u2/createnewstorageacct1.png)

2. **ストレージ アカウントの資格情報**ブレードで、をクリックして**+ 追加**です。

    ![ストレージ アカウントの資格情報を追加します。](./media/storsimple-8000-configure-new-storage-account-u2/createnewstorageacct2.png)

3. **ストレージ アカウントの資格情報を追加**ブレードで、次の手順を実行します。

    1. ように、サービスと同じ Azure サブスクリプションでストレージ アカウントの資格情報を追加することを確認する**現在**が選択されています。

    2. **ストレージ アカウント** ドロップダウン リストで、既存のストレージ アカウントを選択します。

    3. 選択すると、ストレージ アカウントに基づいて、**場所**が表示されます (ここで変更することはできず、グレー)。

    4. 選択**SSL モードを有効にする**デバイスとクラウド間のネットワーク通信をセキュリティで保護されたチャネルを作成します。 無効にする**SSL を有効に**プライベート クラウド内で動作している場合のみです。

        ![ストレージ アカウントの資格情報 ブレードを追加します。](./media/storsimple-8000-configure-new-storage-account-u2/createnewstorageacct3.png)

    5. をクリックして**追加**をストレージ アカウントの資格情報のジョブの作成を開始します。 ストレージ アカウントの資格情報が正常に作成された後通知されます。

        ![ストレージ アカウントの資格情報の通知を成功](./media/storsimple-8000-configure-new-storage-account-u2/createnewstorageacct5.png)

下の一覧で、新しく作成されたストレージ アカウントの資格情報が表示されます**ストレージ アカウントの資格情報**です。

![ストレージ アカウントの資格情報の一覧](./media/storsimple-8000-configure-new-storage-account-u2/createnewstorageacct6.png)
