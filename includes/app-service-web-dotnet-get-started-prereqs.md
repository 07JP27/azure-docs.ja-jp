### <a name="setupdevenv"></a>Visual Studio 2015 と Azure SDK for .NET
このチュートリアルは、Visual Studio 2015 と [Azure SDK for .NET](../articles/dotnet-sdk.md) 2.8.2 以降向けに書かれています。[最新の Azure SDK for Visual Studio 2015 はここからダウンロードしてください](http://go.microsoft.com/fwlink/?linkid=518003)。Visual Studio をまだ持っていない場合は、SDK と共に自動的にインストールされます。

Visual Studio 2013 を持っている場合は、[最新の Azure SDK for Visual Studio 2013 をダウンロード](http://go.microsoft.com/fwlink/?LinkID=324322)できます。一部の画面は画像と異なる場合があります。

> [!NOTE]
