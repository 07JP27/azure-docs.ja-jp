1. Visual Studio の **[ツール]** メニューの **[拡張機能と更新プログラム]** を選択し、**[オンライン]** > **[Visual Studio Marketplace]** の順に選択して、**[Azure Functions Tools for Visual Studio 2017]** を選択します。その後、**[ダウンロード]** をクリックします。
