1. ポータルで VM を選択します。 **[DNS 名]** の下の **[構成]** をクリックします。
   
   ![[DNS 名] の下の [構成] をクリックする](./media/virtual-machines-common-portal-create-fqdn/configure.png)

2. 目的の DNS 名を入力し、**[保存]** を選択します。
   
   ![パブリック IP リソースの DNS 名ラベルを入力](./media/virtual-machines-common-portal-create-fqdn/configure-pane.png)


3. VM の概要ブレードに戻るために、*[パブリック IP アドレス]* ブレードを閉じます。 "*DNS 名*" が表示されていることを確認します。
