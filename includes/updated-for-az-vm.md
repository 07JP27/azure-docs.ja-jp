> [!NOTE]
>
> この記事は、新しい [Azure PowerShell Az モジュール](/powershell/azure/new-azureps-module-az)を使用するように更新されました。 AzureRM では重要なバグ修正の更新が継続されますが、新機能は Az モジュールだけに追加されます。
> * Az モジュールは Cloud Shell 上で既に使用可能であり、AzureRM との互換性が有効になっています。
> * Az モジュールをローカルでインストールするには、[Azure PowerShell のインストール](/powershell/azure/install-az-ps)に関するページを参照してください。
> * Az モジュールをローカルにインストールする場合、`Enable-AzureRmAlias` を実行して、AzureRM 互換性を有効にできます。
