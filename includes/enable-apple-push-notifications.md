
* [アプリのアプリ ID を登録します](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991)。 ワイルドカード アプリ ID ではなく明確なアプリ ID を作成し、**バンドル ID** では、Xcode クイックスタート プロジェクト内に含まれる**バンドル ID** を使用します。 **[プッシュ通知]** チェック ボックスをオンにすることも重要です。
