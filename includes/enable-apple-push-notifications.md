
* [アプリのアプリ ID を登録します](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991)。ワイルドカード アプリ ID ではなく明確なアプリ ID を作成し、**バンドル ID** では、Xcode クイックスタート プロジェクト内に含まれる **バンドル ID** を使用します。**[プッシュ通知]** チェック ボックスをオンにすることも重要です。 

* 次に、[アプリのプッシュ通知を有効にします](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringPushNotifications/ConfiguringPushNotifications.html#//apple_ref/doc/uid/TP40012582-CH32-SW6)。"開発用" または "配布用" の SSL 証明書を作成できます (後で必ず Azure ポータル内の該当するオプションを選択してください)。

* 次に、[アプリ ID によってプッシュ通知が有効になっていることを確認](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringPushNotifications/ConfiguringPushNotifications.html#//apple_ref/doc/uid/TP40012582-CH32-SW8)します。

* 最後に、[Xcode クイックスタート プロジェクト内のプロビジョニング プロファイルを更新](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringPushNotifications/ConfiguringPushNotifications.html#//apple_ref/doc/uid/TP40012582-CH32-SW10)して、[プッシュ通知を有効にするようにプロビジョニング プロファイルが作成または再生成されたことを確認](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringPushNotifications/ConfiguringPushNotifications.html#//apple_ref/doc/uid/TP40012582-CH32-SW12)します。
