# Azure Marketplace の発行元サポートへのアクセス

Microsoft は多種多様な製品とサービスを提供しています。適切でタイムリーな応答を行うには、クエリによって適切なサポート チームを見つけることが重要です。次のシナリオを検討して、クエリで適切なチームを見つけるのに役立ててください。

- Marketplace の発行元であり、まだ VM の開発中である場合は、Microsoft オンボードまたはコンピューティング チームに連絡する必要があります。

- Marketplace の発行元であり、お客様から質問を受けた場合は、お客様に https://ms.portal.azure.com/ のサポート リンクからサポートを要求するように伝えてください。

- Marketplace の発行元であり、アプリケーションまたはサービスに関する質問がある場合は、以下を参照してください。

## この記事では、次の操作方法について説明します。

- Microsoft アカウントへのサインイン
- 発行元サポート ページへの移動
- 問題の種類とカテゴリの選択
- サポートの要求
- 連絡先情報の提供
- 問題の説明

## 1\.Microsoft アカウントへのサインイン

https://publish.windowsazure.com/ に移動し、Microsoft アカウントの資格情報でサインインします

  ![ログイン画面][1]

## 2\.発行元サポート ページへの移動

右上のメニューから、[サポートの要求] を選択します

  ![サポートの要求][2]

## 3\.問題の種類とカテゴリの選択

![問題の種類とカテゴリ][3]

次の問題の種類を選択できます。

  - アカウント管理
  - レポートおよびインサイト
  - VM イメージの認定
  - 顧客の管理
  - プランの管理
  - 支払
  - 販売者ダッシュボード
  - その他

## 4\.サポートの要求

問題の種類とカテゴリを選択したら、[要求の開始] ボタンをクリックします

![サポートの開始][4]

## 5\.連絡先情報の提供と問題の説明

可能な限り詳細に問題を説明し、問題の解決に役立つ可能性のあるすべてのファイル (スクリーン ショット、ネットワークのトレースなど) を提供します。

![問題の説明][5]

## 6\.送信の確認

問題を送信すると、24 時間以内に受信確認が行われ、問題に優先順位と重大度が割り当てられます。サポート エンジニアから成る専門チームによって、サービス レベル アグリーメントの応答時間に従い、解決の支援が行われます。

![確認][6] + 問題について対話する必要がある場合は、すべての通信で確認番号を使用してください。また、問題に関する進捗はいつでも確認できます。

## 次のステップ

特定のサポート問題を解決する方法を確認するには、「[Troubleshooting common issues on the Azure Marketplace (Azure Marketplace での一般的な問題のトラブルシューティング)](marketplace-publishing-support-common-issues.md)」を参照してください。
