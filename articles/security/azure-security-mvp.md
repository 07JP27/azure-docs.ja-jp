# Azure セキュリティの MVP プログラム

Microsoft Most Valuable Professional (MVP) は、Microsoft テクノロジを最大限に活用できるように、他者を献身的に支援した模範的コミュニティ リーダーです。並外れた情熱、実世界の知識、および技術的専門知識を、コミュニティや Microsoft と共有します。

Microsoft Azure では、Azure セキュリティに関する特別な専門知識を持つコミュニティ エキスパートを承認するようになったことを発表いたします。Microsoft MVP は、Azure セキュリティの貢献エリアにおいて、Microsoft Azure の MVP を取得できます。

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

主にテクノロジとそのライフサイクルによって異なることが理由で、MVP になるための基準はありませんが、Microsoft Answers や TechNet、MSDN などのオンライン フォーラム、wiki、オンライン コンテンツ、会議、ユーザー グループ、ポッドキャスト、Web サイト、ブログ、ソーシャル メディア、記事、書籍での貢献度が評価されます。

Azure セキュリティの専門家ですか? 専門家をご存知ですか? [自分自身または他のユーザーを指名して](https://mvp.microsoft.com/Nomination/nominate-an-mvp)、今すぐ Azure セキュリティ MVP になりましょう。
