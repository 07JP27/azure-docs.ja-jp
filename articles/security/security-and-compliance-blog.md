# Azure Security and Compliance のブログ

[Azure Security and Compliance チームのブログ](https://blogs.msdn.microsoft.com/azuresecurity/)では、Azure のセキュリティおよびコンプライアンスに関する最新の技術情報を参照できます。

さらに、Azure Security and Compliance チームのブログでは、[Azure Security Center](https://azure.microsoft.com/documentation/services/security-center/) の技術的なトピックの重要な更新についても検索できます。

Azure Security and Compliance チームのブログでは、技術的な情報だけでなく、Azure のセキュリティとコンプライアンスに関する最新のお知らせも、Azure ブログに投稿されしだい紹介されます。

Azure Security and Compliance チームのブログを確認して、Azure のセキュリティに関するすべてのことの最新情報をご覧ください。
