# <a name="awareness-and-training-at"></a>意識向上とトレーニング (AT)

> [!NOTE]
> この管理策は、NIST および米国商務省により、NIST Special Publication 800-53 Revision 4 の一部として定義されています。 各コントロールのテスト手順とガイダンスについては、NIST 800-53 Rev. 4 を参照してください。

## <a name="nist-800-53-control-at-1"></a>NIST 800-53 コントロール AT-1

#### <a name="security-awareness-and-training-policy-and-procedures"></a>セキュリティの意識向上とトレーニングに関するポリシーと手順

**AT-1** 組織は、目的、スコープ、役割、責任、経営陣のコミットメント、組織エンティティ間の調整、およびコンプライアンスを扱った、セキュリティの意識向上とトレーニングに関するポリシーのほか、セキュリティの意識向上とトレーニングに関するポリシーおよびそれに関連するコントロールの実施を促進する手順を策定および文書化し、(義務: 組織が定めた社員または役職) に周知する。また、セキュリティの意識向上とトレーニングに関する現在のポリシー (義務: 組織が定めた頻度)、および意識向上とトレーニングに関する手順 (義務: 組織が定めた頻度) を見直し、更新する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | このコントロールに対応するには、セキュリティの意識向上とトレーニングに関する、お客様のエンタープライズレベルのポリシーと手順で十分な場合があります。 |
| **プロバイダー (Microsoft Azure)** | 適用外 |


 ## <a name="nist-800-53-control-at-2a"></a>NIST 800-53 コントロール AT-2.a

#### <a name="security-awareness-training"></a>セキュリティ意識向上トレーニング

**AT-2.a** 組織は、新規ユーザーの初期トレーニングの一環として、情報システムのユーザー (マネージャー、上級管理者、契約社員など) に基本的なセキュリティ意識向上トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング プログラムで、すべてのシステム ユーザーに初期トレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ## <a name="nist-800-53-control-at-2b"></a>NIST 800-53 コントロール AT-2.b

#### <a name="security-awareness-training"></a>セキュリティ意識向上トレーニング

**AT-2.b** 組織は、情報システムの変更に応じて必要な場合に、情報システムのユーザー (マネージャー、上級管理者、契約社員など) に基本的なセキュリティ意識向上トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング プログラムで、すべてのシステム ユーザーに対して、必要に応じたトレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用外 |


 ## <a name="nist-800-53-control-at-2c"></a>NIST 800-53 コントロール AT-2.c

#### <a name="security-awareness-training"></a>セキュリティ意識向上トレーニング

**AT-2.c** 組織は以後も (義務: 組織が定義した頻度)、情報システムのユーザー (マネージャー、上級管理者、契約社員など) に対して基本的なセキュリティ意識向上トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング プログラムで、すべてのシステム ユーザーに継続的なトレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ### <a name="nist-800-53-control-at-2-2"></a>NIST 800-53 コントロール AT-2 (2)

#### <a name="security-awareness-training--insider-threat"></a>セキュリティ意識向上トレーニング | 内部関係者の脅威

**AT-2 (2)** 組織は、内部関係者の脅威に関する潜在的な指標の認識と報告について、セキュリティ意識向上トレーニングを用意する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング コンテンツで、内部関係者の脅威の認識に対応できます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ## <a name="nist-800-53-control-at-3a"></a>NIST 800-53 コントロール AT-3.a

#### <a name="role-based-security-training"></a>ロールベースのセキュリティ トレーニング

**AT-3.a** 組織は、セキュリティに関する役割と責任を割り当てられた社員に対して、情報システムへのアクセスの承認前、または割り当て済みの職務の実行前に、ロールベースのセキュリティ トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのトレーニング プログラムで、ロールベースのセキュリティ トレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ## <a name="nist-800-53-control-at-3b"></a>NIST 800-53 コントロール AT-3.b

#### <a name="role-based-security-training"></a>ロールベースのセキュリティ トレーニング

**AT-3.b** 組織は、情報システムの変更による必要に応じて、セキュリティに関する役割と責任を割り当てられた社員にロールベースのセキュリティ トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのトレーニング プログラムで、必要に応じたロールベースのセキュリティ トレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用外 |


 ## <a name="nist-800-53-control-at-3c"></a>NIST 800-53 コントロール AT-3.c

#### <a name="role-based-security-training"></a>ロールベースのセキュリティ トレーニング

**AT-3.c** 組織は以後も (義務: 組織が定めた頻度)、セキュリティに関する役割と責任を割り当てられた社員にロールベースのセキュリティ トレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのトレーニング プログラムで、継続的なロールベースのセキュリティ トレーニングを実施できます。 |
| **プロバイダー (Microsoft Azure)** | 適用外 |


 ### <a name="nist-800-53-control-at-3-3"></a>NIST 800-53 コントロール AT-3 (3)

#### <a name="role-based-security-training--practical-exercises"></a>ロールベースのセキュリティ トレーニング | 実践的な演習

**AT-3 (3)** 組織は、トレーニングの目的を強化する実践的な演習をセキュリティ トレーニングに含める。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様が行うエンタープライズレベルのロールベースのセキュリティ トレーニングに、実践的な演習を含めることができます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ### <a name="nist-800-53-control-at-3-4"></a>NIST 800-53 コントロール AT-3 (4)

#### <a name="role-based-security-training--suspicious-communications-and-anomalous-system-behavior"></a>ロールベースのセキュリティ トレーニング | 疑わしい通信とシステムの異常な動作

**AT-3 (4)** 組織は、疑わしい通信と組織の情報システムにおける異常な動作を認識できるように、(義務: 悪質なコードに関して組織が定めた指標) について社員にトレーニングを実施する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様が用意するエンタープライズレベルのロールベースのセキュリティ トレーニングに、疑わしい通信とシステムの異常な動作の認識を含めることができます。 |
| **プロバイダー (Microsoft Azure)** | 適用外 |


 ## <a name="nist-800-53-control-at-4a"></a>NIST 800-53 コントロール AT-4.a

#### <a name="security-training-records"></a>セキュリティ トレーニングの記録

**AT-4.a** 組織は、基本的なセキュリティ意識向上トレーニングや情報システムに関する特定のセキュリティ トレーニングなど、情報システムに関する個々のセキュリティ トレーニング活動を文書化および監視する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング プログラムに、トレーニング活動の文書化および監視を含めることができます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |


 ## <a name="nist-800-53-control-at-4b"></a>NIST 800-53 コントロール AT-4.b

#### <a name="security-training-records"></a>セキュリティ トレーニングの記録

**AT-4.b** 組織は、(義務: 組織の定めた期間) 個人のトレーニング記録を保持する。

**責任:** `Customer Only`

|||
|---|---|
| **お客様** | お客様のエンタープライズレベルのセキュリティ意識向上トレーニング プログラムは、トレーニング記録の保持に関する要件を満たすことができます。 |
| **プロバイダー (Microsoft Azure)** | 適用されません |
