# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool 更新プログラム リリース 7.1.51023.1 - 11/1/2018

[GA のリリース ノート](threat-modeling-tool-releases-71509112.md)にもともと記載されていたように、Microsoft Threat Modeling Tool に対する更新プログラム (7.1.51023.1) のリリースに伴い、プレビュー バージョン (バージョン 7.1.50911.2 未満のプレビュー クライアント) のユーザーはサポートされる GA リリースにアップグレードする必要があります。 このリリースには、新しい機能や修正プログラムは含まれません。

- プレビュー バージョンのユーザーは、クライアントを開くと、アップグレードが自動的にダウンロードされます。 新しい更新プログラムをインストールしないことを選択した場合、ツールのプレビュー バージョンは閉じられます。
- GA バージョンのツールのユーザーには、アップグレードするかどうかの選択を求めるメッセージが表示されます。
- 2016 バージョンのツールのユーザーには影響はありません。

## <a name="feature-changes"></a>機能の変更

- なし

## <a name="system-requirements"></a>システム要件

- サポートされているオペレーティング システム
  - Microsoft Windows 10
- 必要な .NET のバージョン
  - .NET 3.5.2
- その他の要件
  - ツールとテンプレートの更新プログラムを受け取るには、インターネット接続が必要です。
