# <a name="threat-modeling-tool-releases"></a>Threat Modeling Tool のリリース

Microsoft Threat Modeling Tool は現在、Windows 用に無料で[クリックしてダウンロードできる](https://aka.ms/threatmodelingtool)アプリケーションとしてリリースされています。 この配布のしくみでは、ユーザーがツールを開くたびに、最新の改善とバグの修正をプッシュできます。

## <a name="system-requirements"></a>システム要件

- サポートされているオペレーティング システム
  - Microsoft Windows 10 Anniversary Update 以降
- 必要な .NET のバージョン
  - .NET 4.7.1 以降
- その他の要件
  - ツールとテンプレートの更新プログラムを受け取るには、インターネット接続が必要です。
