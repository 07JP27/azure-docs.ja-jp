# Azure のロード テスト

クラウド ベースのロード テスト サービスでは、Web アプリケーションに対してロード テストを実施できます。Web アプリケーションの容量およびパフォーマンスを明らかにするロード テストを作成するには、Visual Studio を使用します。このとき、Azure VM を使用する必要はありません。ロード テストは、[クラウド ベースのロード テスト サービス][クラウド ベースのロード テスト サービス]で作成できます。
