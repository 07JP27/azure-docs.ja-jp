# Azure Machine Learning ワークスペースの管理 
Microsoft Azure Machine Learning (Azure ML) ワークスペースを管理して、アクセスの管理、運用 Web サービスのデプロイ、Azure ML Web サービスの管理を行うことができます。さらに、[ダッシュボード] タブで、ワークスペースの使用状況の概要を表示したり、ワークスペース情報をひとめで確認したりできます。  

**[Web サービス]** タブで、Azure ML Web サービスを追加、更新、削除できます。  

## ワークスペースの管理ページに移動するには ##
1.	Microsoft Azure のアカウントを使用してサインインします。
2.	Microsoft Azure サービス パネルで、**[MACHINE LEARNING]** をクリックします。
3.	**[機械学習]** リストで、ワークスペースをクリックします。
