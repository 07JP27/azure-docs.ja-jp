# Azure の請求とサブスクリプションの問題についてサポート チケットを作成する方法

この記事では、Azure サポートでチケットを開くときに必要な情報を提供します。

>[AZURE.NOTE]アカウント センターへのアクセス権があるのは Azure アカウント管理者だけです。

1. [Azure サポート](http://azure.microsoft.com/support) Web サイトに移動して、**[サポートの要求]** をクリックします。これ以外に、Microsoft Azure ポータルまたは Azure アカウント センターからインシデントを送信する方法もあります。

 * [Azure ポータル](https://manage.windowsazure.com/)からサポート チケットを送信するには、アカウント名をクリックし、[Microsoft サポートに問い合わせる] をクリックします。
 * [Azure アカウント センター](https://account.windowsazure.com/Subscriptions)からサポート チケットを送信するには、サブスクリプションをクリックし、[Microsoft サポートに問い合わせる] をクリックします。

2. **[サポート チケットの作成]** ウィンドウで、データとサービスを移行するサブスクリプションを選択します。

3. **[サポートの種類]** で **[課金]** を選択します。

4. 地域と言語を選択し、**[チケットの作成]** をクリックします。

5. 次のページの **[問題の種類]** で **[サブスクリプションが無効と表示される]** を選択します。サポート チケットの送信に関してサポートが必要な場合は、**[カスタマー サービスにチャットで質問する]** をクリックして、カスタマー サービス担当者とのライブ チャットをご利用ください。

6. 自分の連絡先情報を確認し、連絡の取りやすい電話番号を指定して、**[続行]** をクリックします。

7. 次に表示されるページで、次の情報を指定します。

 * **[問題の詳細]** で、リクエストに使用するインシデント タイトルを指定し、テキスト ボックスを使用して必須情報を入力します。リクエストについて他にも情報があればテキスト ボックスに追加してください。
 * **[深刻度の設定]** で、リクエストの深刻度を表すオプションを選択します。深刻度の種類の詳細については、[Microsoft Azure のインシデントの深刻度を定義した表](http://support.microsoft.com/gp/AzureSevDetails)を参照してください。
 * **ファイルのアップロード** ツールを使用して関連するドキュメントをリクエストに添付します。

8. Microsoft サービスの契約に同意する場合は、**[送信]** をクリックします。

契約の条項に従って、Azure のサポート担当者から連絡があります。
