# Azure の請求とサブスクリプションの問題についてサポート チケットを作成する方法

この記事では、Azure サポートでチケットを開くときに必要な情報を提供します。

1. [Azure ポータル](https://portal.azure.com/)にサインインします。**[ヘルプとサポート]**、**[新しいサポート要求]** の順に選択します。

	![helpandsupportbutton](./media/billing-how-to-create-billing-support-ticket/helpandsupport.png)
2. **[基本]** ブレードで、問題の種類、サブスクリプション、サポート プランを選択します。
3. **[問題]** ブレードに次の情報を指定します。

 * **深刻度**。要求の深刻度を表すオプションを選択します。深刻度の種類の詳細については、[Microsoft Azure のインシデントの深刻度を定義した表](http://support.microsoft.com/gp/AzureSevDetails)を参照してください。
 * **問題の種類**。問題の種類を選択します。
 * **詳細**。要求に使用するインシデント タイトルを指定し、テキスト ボックスを使用して必須情報を入力します。要求について他にも情報があればテキスト ボックスに追加してください。
 * **期間** (省略可能)。可能であれば、最近発生した問題の日付、時刻、時間帯情報を指定します。
 * **ファイルのアップロード** ツールを使用して関連するドキュメントをリクエストに添付します。
4. **[連絡先情報]** ブレードで、連絡先情報と優先連絡方法を確認し、**[作成]** をクリックします。

契約の条項に従って、Azure のサポート担当者から連絡があります。

サポート要求は Azure サポート Web サイトや Azure アカウント センターからも送信できます。

 * [Azure サポート Web サイト](https://azure.microsoft.com/support/options/)からサポート チケットを送信するには、**[サポートの要求]** をクリックします。
 * [Azure アカウント センター](https://account.windowsazure.com/Subscriptions)からサポート チケットを送信するには、サブスクリプションを選択し、**[Microsoft サポートに問い合わせる]** をクリックします。

 >[AZURE.NOTE]アカウント センターへのアクセス権があるのは Azure アカウント管理者だけです。サブスクリプションのアカウント管理者を確認する方法の詳細については、[よく寄せられる質問](billing-subscription-transfer.md#faq)を参照してください。
