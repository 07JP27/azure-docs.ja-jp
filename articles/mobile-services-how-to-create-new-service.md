# 新しいモバイル サービスを作成する方法

このトピックでは、[Azure 管理ポータル][Azure 管理ポータル]を使用して、サーバー側ビジネス ロジックに JavaScript を使用する新しい JavaScript バックエンドを Azure Mobile Services で作成する方法について説明します。モバイル サービス バックエンドと、新しいモバイル サービスを使用するモバイル クライアント アプリケーションの両方を作成するには、代わりに「[モバイル サービスの使用][モバイル サービスの使用]」を参照してください。

> [WACOM.NOTE] このチュートリアルを完了するには、Azure Mobile Services の機能を有効にした Azure アカウントが必要です。アカウントがない場合は、無料の試用アカウントを数分で作成することができます。詳細については、[Azure の無料評価版サイト][Azure の無料評価版サイト]を参照してください。
