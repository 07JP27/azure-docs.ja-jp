# 仮想マシンの Hadoop サンドボックスを使用した Hadoop エコシステム入門

Hortonworks の Hadoop サンドボックスを仮想マシンで設定し、Hadoop エコシステムについて学ぶことができます。Hadoop、Hadoop 分散ファイル システム (HDFS)、ジョブの送信などについて理解できるように、サンドボックスにはローカル開発環境が用意されています。

Hortonworks Hadoop サンドボックスの使用については、「[Hortonworks Sandox (Hortonworks サンドボックス)](http://hortonworks.com/downloads/#sandbox)」の「**Hortonworks Sandbox on a VM (仮想マシンの Hortonworks サンドボックス)**」セクションをご覧ください。このサンドボックスで使用できるチュートリアルを実行して、HDFS、クラスターへのジョブの送信方法、クラスターで実行されているジョブの追跡方法などについて理解することをお勧めします。

Hadoop について理解できたら、HDInsight クラスターを作成して、Azure で Hadoop を使い始めることができます。使用方法の詳細については、[HDInsight での Hadoop の使用](hdinsight-hadoop-linux-tutorial-get-started.md)に関するページをご覧ください。
