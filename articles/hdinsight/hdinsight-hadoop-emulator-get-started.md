# Hadoop サンドボックスを使用した Hadoop エコシステム入門

仮想マシンで (Hortonworks から) Hadoop サンドボックスを設定し、Hadoop エコシステムについて説明します。Hadoop、Hadoop 分散ファイル システム (HDFS)、ジョブの送信などについて理解できるように、サンド ボックスにはローカル開発環境が用意されています。Hortonworks Hadoop サンド ボックスを開始するには、「[Hortonworks Sandox (Hortonworks サンドボックス)](http://hortonworks.com/downloads/#sandbox)」の「**Hortonworks Sandbox on a VM (VM 上の Hortonworks サンド ボックス)　**」セクションをご覧ください。このサンド ボックスで使用できるチュートリアルを実行して、HDFS、クラスターへのジョブの送信方法、クラスターで実行されているジョブの追跡方法などについて理解することをお勧めします。

Hadoop について理解できたら、HDInsight クラスターを作成して、Azure で Hadoop を使い始めることができます。開始方法の詳細については、[HDInsight での Hadoop の使用](hdinsight-hadoop-linux-tutorial-get-started.md)に関するページをご覧ください。
