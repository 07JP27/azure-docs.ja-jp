# <a name="azure-china-21vianet"></a>Azure China 21Vianet

21Vianet によって運用されている Microsoft Azure は、中国の規制要件に準拠するように設計され、拡大し続ける統合クラウド サービスのコレクションを含むクラウド プラットフォームです。 開発者や IT プロフェッショナルがアプリケーションのビルド、デプロイ、管理に使用するサービスです。 Azure China 21Vianet は、中国本土内にある物理的に分離されたクラウド サービスのインスタンスであり、Shanghai Blue Cloud Technology Co., Ltd. ("21Vianet") によって独立して運用され、処理されています。

Azure China 21Vianet は、中国の規制でそのアーキテクチャに基づいて構築されたすべてのシステムとアプリケーションに対して、必要とされる世界規模のセキュリティおよび<a href="https://www.trustcenter.cn/en-us/">コンプライアンス サービス</a>を提供します。 Microsoft の商用クラウド サービスは、中国政府によって定められたポリシーとデータ処理規制を順守しています。
