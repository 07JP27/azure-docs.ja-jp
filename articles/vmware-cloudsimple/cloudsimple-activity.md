# <a name="activity-management-overview"></a>アクティビティ管理の概要

CloudSimple は、プライベート クラウドの機能に影響する可能性のあるすべてのアクティビティを追跡します。 このようなアクティビティには、アラート、イベント、タスク、監査アクティビティなどが含まれます。 [[アクティビティ]](https://docs.azure.cloudsimple.com/activity/) ページには、現在のすべてのアクティビティがまとめられており、詳細情報をドリルダウンできます。

## <a name="events"></a>events

イベントは、CloudSimple ポータルでユーザーおよびシステム アクティビティを追跡します。  イベントは特定のリソースに関連するアクティビティと影響の重大度を示します。  CloudSimple ポータルからイベントを表示できます。

## <a name="alerts"></a>アラート

アラートは、CloudSimple 環境での重要なアクティビティの通知です。  請求やユーザー アクセスに影響するイベントは、アラートとして表示されます。  発生したアラートは CloudSimple から確認できます。

## <a name="tasks"></a>タスク

タスクは、ユーザー操作を追跡し、完了するまでに 30 秒以上かかります。  タスクの進行状況は CloudSimple ポータルから監視できます。  完了したタスクには、そのタスクが完了するまでにかかった合計時間が示されます。

## <a name="audit"></a>Audit

監査ログは、ユーザーが実行した操作を追跡します。  このログには、操作についてユーザーが指定したパラメーターが含まれます。  監査ログを使用して、すべてのユーザーのユーザー アクティビティを監視できます。
