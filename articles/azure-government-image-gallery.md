#  Microsoft Azure の政府のイメージ ギャラリー

<p> Microsoft Azure の政府のイメージ ギャラリーは、パブリックの Azure ポータルには、類似のエクスペリエンスです。顧客はマイクロソフトとパートナー、構築済みイメージを展開または独自の Vhd をアップロードすることができます。これにより、柔軟に必要な場合は、独自の標準化されたイメージを展開します。
