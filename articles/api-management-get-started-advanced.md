# Azure API Management の詳細な構成について

このセクションでは、API Management の詳細な構成とタスクに関するチュートリアルを紹介しています。

-   [Azure API Management での開発者ポータルのカスタマイズ][] - 開発者ポータルのルック アンド フィールをお客様のブランドに合わせてカスタマイズする方法についてのチュートリアルです。
-   [How to use the API Inspector to trace calls in Azure API Management (Azure API Management の API Inspector を使用して呼び出しをトレースする方法)][] - API Inspector を使用して操作の呼び出しをトレースし、情報を収集する方法についてのチュートリアルです。
-   [Azure API Management で成果物を作成して詳細設定を行う方法][] - レート制限ポリシーやクォータ ポリシーなど、成果物の詳細な設定を行う方法についてのチュートリアルです。レート制限を超えるまで操作を繰り返し呼び出すデモを行っています。
-   [How to delegate user registration and product subscription (ユーザーの登録と成果物のサブスクリプションを委任する方法)][] - 開発者のサインイン/サインアップおよび成果物のサブスクリプション処理を、開発者ポータルの組み込みの機能ではなく、お客様の既存の Web サイトを使用して行う方法についてのチュートリアルです。
