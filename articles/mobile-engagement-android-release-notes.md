#リリース ノート

##3.0.0 (2015 年 2 月 17 日)

-   Azure Mobile Engagement の初版リリース
-   appId 構成が接続文字列構成に置き換わりました。
-   任意の XMPP エンティティから任意の XMPP メッセージを送受信する API を削除しました。
-   デバイス間でメッセージを送受信する API を削除しました。
-   セキュリティが強化されました。
-   Google Play と SmartAd の追跡機能が削除されました。
