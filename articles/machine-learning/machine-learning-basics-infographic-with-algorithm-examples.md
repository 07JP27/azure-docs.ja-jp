# インフォグラフィックのダウンロード: Machine Learning の基礎とアルゴリズムの使用例

機械学習の基礎をわかりやすく概説したインフォグラフィックをぜひダウンロードしてください。広く使われているアルゴリズムの中から、機械学習で扱う一般的な問題への答えを導き出すために必要なアルゴリズムを見極めることができます。機械学習の初心者向けに、使用するアルゴリズムの選択や目的を理解するうえで役立つ使用例へのリンクが用意されています。

## 広く使われているアルゴリズムに対応する Machine Learning Studio

[Azure Machine Learning Studio](https://studio.azureml.net/) には、予測分析の問題解決に使用できる多くの機械学習アルゴリズムが付属しています。以下に示したダウンロード可能なインフォグラフィックでは、**回帰**、**異常検出**、**クラスタリング**、**分類**という 4 種類の機械学習アルゴリズムを使って機械学習の疑問に答えます。

[Cortana インテリジェンス ギャラリー](https://gallery.cortanaintelligence.com/)には、これらのアルゴリズムをいかに実世界のソリューションに適用するかを示したサンプルが豊富に用意されています。インフォグラフィックに埋め込まれているリンクから、それぞれのアルゴリズムの実例を参照できます。
