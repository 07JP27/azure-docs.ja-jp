# Machine Learning に関するクラウド データ科学プロセス

このクラウド データ科学プロセスのガイドには、データ科学および Azure Machine Learning を使用した操作可能なモデルを構築するための Azure プラットフォームの使用方法が示されています。マップでは、さまざまなソースからのデータの取り込み、Azure Machine Learning でそれらのデータを使用するための準備、エンド ユーザー アプリケーションによって (プログラム的またはそれ以外の方法で) 使用される操作可能なモデルを作成するための Azure Machine Learning の使用などの一般的なデータ科学タスクについて順を追って説明します。マップには、一般的なエンド ツー エンドのデータ科学の演習に関連する主な手順が示されていますが、すべての手順が必要なわけではありません。また、順番は、データの場所、サイズ、および複雑さによって異なることがあります。

**Azure での高度なデータ処理**というタイトルの手順は、2 番目のマップにリンクされています。そのマップから戻る場合は、Azure Machine Learning Studio で処理されたデータを取り込み、モデルを構築およびデプロイする時点から続行されます。
