# 学習ガイド: Azure での高度なデータ処理 

このマップでは、Azure Machine Learning の実験で使用されるデータを準備する手順について説明します。機械学習用に準備するために、データを移動、処理、サンプリングする方法を示しています。ここで説明されているデータの準備プロセスは、「[Machine Learning に関するクラウド データ科学プロセス](machine-learning-data-science-how-to-create-machine-learning-service.md)」に記載されているマップのステップです。ADAPT ガイドでは、Azure Machine Learning での予測モデルの構築とデプロイに伴って一般的に生じるデータ科学タスクについて取り上げています。
