# Azure Search による最初の検索ソリューションの作成

## 近日対応予定

ご不便をおかけして申し訳ございません。Azure Search の機能を示すサンプル アプリケーションは現在最終的な調整を行っています。数日中に入手可能になります。準備が整いましたら、サンプル コードをダウンロードし、説明に従って Azure Search サービス内で実行することができます。

現在、いくつかの手順を実行できます。いくつかの項目をダウンロードしてサービスを構成することにより、早く開始できます。

-   Visual Studio 2013 Express for Web
-   Azure Search プレビューを構成する

### Visual Studio 2013 Express for the Web をダウンロードする

サンプル アプリケーションは、Visual Studio 2013 で作成されています。既に既存のエディションの Visual Studio 2013 がある場合は、それを使用してください。そうでない場合は、マイクロソフトから無料版をダウンロードできます。

[ダウンロード センター][]で、Visual Studio 2013 Express for the Web を入手してください。

### 無料の検索サービスを構成する

このチュートリアル用に無料サービスを使用することをお勧めします。これは制限されたバージョンの検索であり、すべての登録者が追加コストなしで使用できます。サブスクリプションがない場合でも、評価版にサインアップできます。

サービスの設定の詳細については、「[Azure プレビュー ポータルで Search を構成する][]」を参照してください。

この説明には、実際に必要な手順よりも多くの手順が示されています。無料サービスが作成されたら、その段階で手順を終了してかまいません。このチュートリアルを完了するには、サービス URL と API キーが必要です。
