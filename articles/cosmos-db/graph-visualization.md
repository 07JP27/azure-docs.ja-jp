# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Azure Cosmos DB Gremlin API でデータ可視化ソリューションを使用してグラフ データを視覚化します

以下のソリューションは、[Apache Tinkerpop コミュニティ](http://tinkerpop.apache.org/#poweredby)によって Gremlin グラフの視覚化用に推奨されています。

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious ロゴ](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise では、グラフのテクノロジとデータ視覚化を使用することで、複雑なデータセットがインタラクティブなビジュアル ネットワークになります。 そのプラットフォームによってデータ ソースに接続することで、調査担当者は、何十億ものエンティティとリレーションシップ間をシームレスに移動できます。 その結果は、クエリやテーブルを操作することなく疑わしい関係を検出する新しい機能です。

Linkurious Enterprise のインタラクティブな視覚化インターフェイスによって、複雑な接続先データを調査するための簡単な方法が提供されます。 特定のエンティティを検索し、接続を展開して隠れた関係を明らかにし、選択したレイアウトを適用して複雑なネットワークを解決できます。 Linkurious は、Azure Cosmos DB Gremlin API 互換になりました。 これはエンドツーエンドのグラフ視覚化シナリオに適しており、ユーザー インターフェイスからの読み取りと書き込みの機能がサポートされています。 

![Linkurious のデモ](./media/graph-visualization/linkurious-demo.gif)

**次の手順:** [Azure Cosmos DB を使用した Linkurious のデモ](https://linkurio.us/contact/)を要求できます。

## <a name="cambridge-intelligence"></a>Cambridge Intelligence

![Cambridge Intelligence ロゴ](./media/graph-visualization/ci-logo.png)


Cambridge Intelligence のグラフ視覚化ツールキットで Azure Cosmos DB がサポートされるようになりました。
- [JavaScript 開発者向けの KeyLines](https://cambridge-intelligence.com/keylines/) 
- [React 開発者向けの Re-Graph](https://cambridge-intelligence.com/regraph/)

![Cambridge Intelligence のデモ](./media/graph-visualization/ci-demo-2.gif)

それらのツールキットを使用すると、ご自身のユース ケースに合わせてハイ パフォーマンスなグラフ視覚化および分析アプリケーションを設計できます。 強力な WebGL 描画と注意深く作成されたコードを利用して、高速で洞察に富んだ視覚化エクスペリエンスがユーザーに提供されます。

あらゆるブラウザー、デバイス、サーバー、データベースと互換性があり、ステップバイステップのチュートリアル、完全にドキュメント化された API、インタラクティブなデモが用意されています。
