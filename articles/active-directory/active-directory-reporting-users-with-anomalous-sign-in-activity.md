# 異常なサインイン アクティビティがあるユーザー
これは、次のレポートの不審なサインインを組み合わせた統合レポートです。<ul><li>不明なソースからのサインイン</li> <li>複数回失敗した後のサインイン</li> <li>複数の地域からのサインイン</li> <li>疑わしいアクティビティが発生している IP アドレスからのサインイン</li> <li>感染している可能性があるデバイスからのサインイン</li> <li>不規則なサインイン アクティビティ</li> </ul>
