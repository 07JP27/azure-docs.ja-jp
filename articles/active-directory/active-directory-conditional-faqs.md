# 条件付きアクセスのよく寄せられる質問

**Office 365 のモバイル デバイス管理を使用している場合、Azure AD でデバイス条件を使用できますか。**
 
O365 の MDM では、すべてのデバイス ポリシーが Office 365 の一部として保持されます。Office 365 で設定されているポリシーと Azure AD で設定されているポリシーの競合を避けるために、O365 の MDM を使用している場合、現時点では Azure AD でのこれらのポリシーの設定は許可されていません。
