# Azure AD Privileged Identity Management の使用

## Privileged Identity Management 拡張機能の開始

1. Azure Active Directory 内にディレクトリを作成します。
2. 数名のグローバル管理者を含むユーザーを新しいディレクトリに追加します。

    >[AZURE.NOTE] ユーザーそれぞれに職場または学校のアカウントが設定されていることを確認します。現時点では、Microsoft アカウントで Azure AD PIM にサインアップすることはできません。

3. そのディレクトリのグローバル管理者として、[Azure ポータル](https://portal.azure.com/)にサインインします。
4. Azure ポータルの右上隅にある自分のユーザー名をクリックします。
5. 先ほど作成したディレクトリを選択します。
6. 左側のナビゲーションの **[新規]** アイコンをクリックします。
7. [作成] メニューから **[セキュリティ + ID]** 選択します。
8. **[Azure AD Privileged Identity Management]** を選択します。
9. **[ダッシュボードにピン留めする]** チェック ボックスはオンのままで **[作成]** ボタンをクリックします。Privileged Identity Management ダッシュボードが開きます。
