# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>マイ プロファイル (プレビュー) ポータルでご自分の最近のサインイン アクティビティを表示する
**マイ プロファイル** ポータルの **[最近のアクティビティ]** ページから、ご自分の職場または学校アカウントを使用して行った最近のサインイン アクティビティをすべて表示できます。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>最近のサインイン アクティビティを表示する
1. ご自分の職場または学校アカウントでサインインした後、 https://myprofile.microsoft.com/ ページに移動します。

2. 左側のナビゲーション ウィンドウから **[最近のアクティビティ]** を選択するか、**[最近のアクティビティ]** ブロックから **[アクティビティの確認]** リンクを選択します。

    ![[最近のアクティビティ] リンクが強調表示されている [マイ プロファイル] ページ](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. サインイン項目を展開し、それぞれに覚えがあることを確認します。 覚えがないサインイン項目を見つけた場合は、お使いのアカウントが侵害されたと仮定して、お使いのパスワードを変更することを強くお勧めします。

    ![サインインの詳細が展開されている [最近のアクティビティ] ページ](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >アカウントが侵害されたという確信がある場合は、「[侵害された Microsoft アカウントにアクセスする方法](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account)」を参照して、パスワードをリセットするか、アカウントのブロックを解除します。

## <a name="next-steps"></a>次の手順

- 最近のサインイン アクティビティを確認した後、ご自分の[セキュリティ情報](user-help-security-info-overview.md)や、接続されている[デバイス](myprofile-portal-devices-page.md)または[組織](myprofile-portal-organizations-page.md)の表示や管理を行ったり、組織で[ご自分のデータがどのように使用されているか](myprofile-portal-privacy-page.md)を確認したりできます。
