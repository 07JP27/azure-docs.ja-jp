次に、Azure AD Connect の使用、および Azure AD とオンプレミスのディレクトリとの統合に関する他のリソースの一覧を示します。

>[AZURE.NOTE]Azure AD Connect には、以前 Dirsync と AAD Sync としてリリースされたコンポーネントと機能が組み込まれています。これらのツールは、長期にわたって個別にリリースされています。すべての将来の機能強化は Azure AD Connect への更新に含まれるため、いつでも最新の機能の入手場所が把握できます。

## ドキュメント

[シングル サインオンを使用した DirSync](https://msdn.microsoft.com/library/azure/dn441213.aspx) - AD FS を使用して、オンプレミスの Active Directory 環境と Azure AD とのフェデレーションを行い、シングル サインオンを実現します。

[シングル サインオンを使用したマルチ フォレストのディレクトリ同期のシナリオ](https://msdn.microsoft.com/library/azure/dn510976.aspx) -複数のディレクトリを Azure AD と統合します。

[Azure AD Connect Health](https://msdn.microsoft.com/library/azure/dn906722.aspx) - オンプレミスの AD FS インフラストラクチャの正常性を監視します。
