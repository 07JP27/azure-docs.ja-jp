# <a name="box-and-azure-active-directory"></a>Box と Azure Active Directory

事前構築済みの統合によって、組織全体での Box の導入を加速させ、ユーザーの導入を促進します。 Box を Azure Active Directory (Azure AD) に接続することによって、ユーザーがすばやく共同作業できるようにします。

![Box アプリのロゴを表示する](./media/box.png)

> [!div class="nextstepaction"]
> [Azure AD で Box を試す >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Box と Azure AD で行えること

- **容易なシングル サインオン** - ユーザーが各自の Azure AD アカウントを使用して Box にサインインできるようにすることで、複数のユーザー名とパスワードを排除し、アクセスを簡素化します。
- **ユーザー プロビジョニングの自動化** - ユーザーが組織に加入、組織を移動、または組織から離脱するときの、Box へのユーザー アカウントの作成、メンテナンス、および削除を自動化します。
- **アクセスのセキュリティ保護、管理、監視** - 条件付きアクセス ポリシーを適用することによって Box へのアクセスをセキュリティで保護し、アクセス レビューによってアクセスを管理し、サインイン ログによってアクセスを監視します。

## <a name="get-started"></a>作業開始

簡単な構成のガイドを使用して、Box の[ユーザー プロビジョニング](https://aka.ms/BoxSSODoc)と[シングル サインオン](https://aka.ms/BoxProvisioningDoc)を設定します。
