# Azure Active Directory とは何ですか。


###このトピックに関するその他の記事
[Azure AD とは](active-directory-whatis.md)<br> [どのよう動作しますか。](active-directory-works.md)<br> [開始](active-directory-get-started.md)<br> [次のステップ](active-directory-next-steps.md)<br> [詳細](active-directory-learn-map.md)

Azure Active Directory (Azure AD) は、企業がクラウドとオンプレミス環境の両方で ID とアクセスを簡単に管理できるサービスです。ユーザーは、iOS、Mac OS X、Android、Windows デバイスなど、お気に入りのデバイスから、1 つの職場または学校のアカウントを使用して、クラウドやオンプレミスの任意の Web アプリケーションにシングル サインオンできます。組織は、統合型の Multi-Factor Authentication を使用してオンプレミスとクラウド両方の機密データやアプリケーションを保護し、安全なローカル アクセスとリモート アクセスを確保できます。また、インフォメーション ワーカーが単一の組織アカウントを使用して安全かつ一貫した方法で企業のリソースにアクセスできるように、オンプレミスのディレクトリを拡張できます。Azure AD は、コストを削減し、セキュリティを強化するための包括的なレポート、分析、およびのセルフ サービス機能も提供しています。Azure AD が備えるエンタープライズ レベルの機能と SLA により、途切れることなくビジネスを継続できます。

![][1]


> [AZURE.NOTE]Azure Active Directory を使用するには、Azure のアカウントが必要です。アカウントを持っていない場合は、[無料の Azure アカウントにサインアップ](http://azure.microsoft.com/pricing/free-trial/)できます。
