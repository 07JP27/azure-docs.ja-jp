# Azure AD Connect Sync: 同期オプションのカスタマイズ

Azure Active Directory Connect Synchronization Services (Azure AD Connect Sync) は、オンプレミスの環境とクラウドの Azure AD 間で ID データの同期に関連するすべての操作を処理する Azure AD Connect のメイン コンポーネントです。同期の観点から、Azure AD Connect Sync は、Active Directory Connector で構成された DirSync、Azure AD Sync、Forefront Identity Manager の後継となります。

<center>![Azure AD Connect Sync](./media/active-directory-aadconnectsync-whatis/sync01.png) </center>
