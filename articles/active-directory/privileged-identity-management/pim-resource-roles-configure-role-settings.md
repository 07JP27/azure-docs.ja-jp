# <a name="privileged-identity-management---resource-roles---configure"></a>Privileged Identity Management - リソース ロール - 構成

ロール設定を構成すると、PIM 環境での割り当てに適用される既定値が定義されます。 リソースに対してこれらを定義するには、左側のナビゲーションから [ロールの設定] タブを選択するか、任意のロールの操作バーから [ロールの設定] ボタンを選択して、現在のオプションを表示します。


## <a name="select-a-role"></a>ロールを選択する
変更するロールを選択します。 次に、[編集] を選択して、既存の設定を変更します。
