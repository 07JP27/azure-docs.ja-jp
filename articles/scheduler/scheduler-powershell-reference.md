# Scheduler PowerShell Cmdlets Reference (Scheduler PowerShell コマンドレット リファレンス)

次のテーブルでは、Azure Scheduler の主なコマンドレットを説明し、各コマンドレットのリファレンス ページへのリンクを示します。

|コマンドレット|コマンドレットの説明|
|---|---|
|**[Get-AzureSchedulerJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722516.aspx)**|Scheduler のジョブや特定の Scheduler ジョブの一覧を取得します。|
|**[Get-AzureSchedulerJobCollection](https://msdn.microsoft.com/ja-jp/library/azure/dn722471.aspx)**|Scheduler ジョブ コレクションを取得します。|
|**[Get-AzureSchedulerJobHistory](https://msdn.microsoft.com/ja-jp/library/azure/dn722514.aspx)**|Scheduler のジョブの履歴を取得します。|
|**[Get-AzureSchedulerLocation](https://msdn.microsoft.com/ja-jp/library/azure/dn722505.aspx)**|使用可能な Scheduler の場所を取得します。|
|**[New-AzureSchedulerHttpJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722492.aspx)**|HTTP アクションの Scheduler ジョブを作成します。|
|**[New-AzureSchedulerJobCollection](https://msdn.microsoft.com/ja-jp/library/azure/dn759640.aspx)**|Scheduler ジョブ コレクションを作成します。|
|**[New-AzureSchedulerStorageQueueJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722518.aspx)**|Storage アクションの Scheduler ジョブを作成します。|
|**[Remove-AzureSchedulerJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722477.aspx)**|Scheduler ジョブを削除します。|
|**[Remove-AzureSchedulerJobCollection](https://msdn.microsoft.com/ja-jp/library/azure/dn722530.aspx)**|Scheduler ジョブ コレクションを削除します。|
|**[Set-AzureSchedulerHttpJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722474.aspx)**|HTTP アクションの Scheduler ジョブを更新します。|
|**[Set-AzureSchedulerJobCollection](https://msdn.microsoft.com/ja-jp/library/azure/dn759626.aspx)**|Scheduler ジョブ コレクションを更新します。|
|**[Set-AzureSchedulerStorageQueueJob](https://msdn.microsoft.com/ja-jp/library/azure/dn722476.aspx)**|Storage アクションの Scheduler ジョブを更新します。|
