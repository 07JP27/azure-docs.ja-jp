### <a name="make-a-code-change"></a>コードに変更を加える
`mywebapi` の VS Code ウィンドウに移動し、`server.js` で既定の GET `/` ハンドラーのコードを編集します。次に例を示します。

```javascript
app.get('/', function (req, res) {
    res.send('mywebapi now says something new');
});
```
