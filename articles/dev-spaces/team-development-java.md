### <a name="make-a-code-change"></a>コードに変更を加える
`mywebapi` の VS Code ウィンドウに移動し、`String index()` メソッドのコードを編集します。次に例を示します。

```java
@RequestMapping(value = "/", produces = "text/plain")
public String index() {
    return "Hello from mywebapi says something new";
}
```
