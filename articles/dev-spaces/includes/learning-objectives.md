このガイドでは、以下の方法について説明します。

- 開発用に最適化された Kubernetes ベースの環境 (_開発空間_) を Azure に作成する。
- VS Code とコマンド ラインを使用して、コンテナー内のコードを繰り返し開発する。
- チーム環境でコードを生産的に開発してテストする。
