# <a name="cloud-solution-providers"></a>クラウド ソリューション プロバイダー

最小限のマーケティングと販売するための投資で、クラウド ソリューション プロバイダー (CSP) パートナーが対応している Microsoft の非常に多くの正規のお客様にソフトウェア オファーを提供できます。  発行元は、[Microsoft のネットショップ](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace)を通してオファーを一般提供することに加え、CSP パートナー チャネルを活用できるようになりました。

発行元は、CSP パートナー チャネルで利用できる新しいまたは既存のオファーのオプトインを選択できます。 この選択によって、CSP パートナーによる製品の販売とお客様向けのバンドル ソリューションの作成を実行できます。

発行元は、エンド ユーザーへの Break-Fix サポートと、CSP パートナーまたは顧客、あるいはその両方がサポートを得るために連絡するための仕組みを提供する責任があります。 ユーザー ドキュメント、トレーニング、サービスの正常性/停止通知 (該当する場合) を CSP パートナーに提供して、CSP パートナーが顧客からのレベル 1 のサポートを処理する体制を整えることができるようにすることをお勧めします。

次の表に、CSP パートナー チャネルでオプトインできるオファーの種類と価格モデルを示します。

| **オファリング タイプ**    | **BYOL**  |  **Free** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| VM  | はい | はい | はい |
| Containers | 該当なし | はい | 該当なし |
| ソリューション テンプレート | 該当なし | はい | 該当なし |
| Managed Applications | 該当なし | はい | はい |
| IT ソリューション SaaS (サイト単位の価格) | 該当なし | はい | はい |
|   |   |   |

## <a name="how-to-opt-in"></a>オプトイン方法

CSP パートナー チャネルのオプトイン設定は、クラウド パートナー ポータルのオファー作成エクスペリエンス内の [Marketplace] タブまたは [Storefront] タブにあります。

![CSP オプトイン エクスペリエンス](media/marketplace-publishers-guide/csp-opt-in.png)
