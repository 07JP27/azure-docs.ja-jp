# <a name="azure-appsource-and-marketplace-review-policies"></a>Azure AppSource および Marketplace のレビュー ポリシー

Azure AppSource と Marketplace についての評価およびレビューのポリシーを次に示します。

* AppSource および Azure Marketplace のレビューで、Microsoft は次の行動やコンテンツを許容しません。
    * ボット/サイバー攻撃
    * プライバシー コンプライアンスの違反
    * スパム コンテンツ
    * 詐欺コンテンツ
    * 不快感を与えるコンテンツ: わいせつ、低俗、または不快感を与える言語またはジェスチャ
    * 不正なコンテンツ
    * 侮辱的なコンテンツ、憎しみに満ちたコンテンツ、あるいは威嚇的なコンテンツ
    * 中傷的なコンテンツ
    * 同じコンテンツや類似したコンテンツを繰り返し送信すること
    * 他のアプリやサービスの宣伝を含む広告

* Microsoft では、ユーザーによって送信されたレビュー関連のコンテンツを、理由の如何を問わず削除する権利を保有します。
* Microsoft は、理由の如何を問わずユーザーがレビュー関連のコンテンツを送信できないようにする権利を保有します。
* パブリッシャーは、自分のアプリケーションに関連するレビュー関連のコンテンツの削除に通知を受け取りません。
* ユーザーは、ユーザーが送信したレビュー関連のコンテンツの削除時に通知を受け取りません。
* Microsoft は、レビュー関連のコンテンツの削除に関連して説明を提供する義務を負いません。
* Microsoft は、削除したレビュー関連のコンテンツを復元する要求に対処しません。

## <a name="review-guidelines-for-customers"></a>顧客のためのレビューに関するガイドライン

* [AppSource および Azure Marketplace のレビュー ポリシー](./rating-review-policies.md)に違反する可能性があるコンテンツを送信しないでください。
* メール アドレス、実際の住所、電話番号など個人を特定できる情報 (PII) を含めないでください。
* レビューは明確で読みやすく、かつ有益なものにしてください。
* レビューで表現されている視点がバランスの取れた中立的な視点となるようにしてください。
* 公開フォーラムに投稿しているのだということを忘れず、それに応じて行動してください。
