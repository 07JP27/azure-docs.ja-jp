# <a name="azure-application-offer"></a>Azure アプリケーション プラン

<table> <tr> <td>このセクションでは、新しいマネージド Azure アプリケーション プランを <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a> に公開する方法について説明します。  各マネージド アプリケーションには、アプリケーションで使用されるすべての技術資産を定義する Azure Resource Manager テンプレートが 1 つ含まれています。このテンプレートには通常、1 つ以上の仮想マシンとその他のサポートする Azure ベースまたは Web ベースのサービスが含まれています。 </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>公開の概要

次のビデオ「[Building Solution Templates, and Managed Applications for the Azure Marketplace](https://channel9.msdn.com/Events/Build/2018/BRK3603)」 (Azure Marketplace 用にソリューション テンプレートとマネージド アプリケーションをビルドする) は、Azure Resource Manager を作成して Azure アプリケーション ソリューションを定義する方法と、その後でアプリ オファーを Azure Marketplace に公開する方法に関する概要です。

> [!VIDEO https://channel9.msdn.com/Events/Build/2018/BRK3603/player]


## <a name="publishing-process"></a>公開プロセス

次の記事では、マネージド Azure アプリケーションを作成して公開するプロセスを説明します。

1. [Microsoft デベロッパー センターでアカウントを登録](./cloud-partner-portal-dev-center-accounts-registration.md)して、Azure Marketplace への公開の承認を得ます。
2. Azure Marketplace に[新しい Azure アプリケーション プランを公開](./cloud-partner-portal-managed-app-publish.md)します。
3. [プランを公開します](./cloud-partner-portal-managed-app-offer-go-live.md)。 


## <a name="next-steps"></a>次の手順

次に、このプランの種類に必要な要件を確認し、[Azure マネージド アプリケーション](./cloud-partner-portal-managed-app-publish.md)に関するページで詳しく説明されているように、公開プロセスを開始する必要があります。
