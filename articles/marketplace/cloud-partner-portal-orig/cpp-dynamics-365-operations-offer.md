# <a name="dynamics-365-for-finance-and-operations-offer"></a>Dynamics 365 for Finance and Operations プラン

<table> <tr> <td>このセクションでは、Dynamics 365 for Finance and Operations ソリューションを Microsoft <a href="https://appsource.microsoft.com">AppSource Marketplace</a> に公開する方法について説明します。 Microsoft の <a href="https://dynamics.microsoft.com/finance-and-operations">Dynamics 365 for Finance and Operations</a> は、高度な財務、経営、製造、およびサプライ チェーン管理をサポートするエンタープライズ リソース プランニング (ERP) サービスです。 </td> <td><img src="./media/publish_d365_new_offer/dynfinance-icon1.png"  alt="Dynamics 365 for Finance and Operations icon" /></td> </tr> </table>


## <a name="publishing-process"></a>公開プロセス

Dynamics 365 for Finance and Operations アプリを開発して AppSource に公開するには、次の手順を使用します。

1. 前提条件として、[Dev Center Certified](https://dev.windows.com/en-us/registration?accountProgram=Azure) であることが必要です。
2. コードをアップグレードします。
3. [ライフサイクル サービス](https://docs.microsoft.com/dynamics365/unified-operations/dev-itpro/lifecycle-services/lcs) (LCS) を使用してデータをビルド、パッケージング、およびデプロイします。
4. 実装方法を LCS で作成します。
5. ビジネス プロセスを LCS で定義します。
6. Microsoft Cloud パートナー ポータルでマーケティング コンテンツを作成します。
7.  アプリを当社チームで検証してもらいます。 ガイダンスについては、[Finance and Operations 用アプリケーションを検証する](https://docs.microsoft.com/dynamics365/operations/dev-itpro/lcs-solutions/app-validation-lcs-solutions)をご覧ください。
8. プランを Microsoft Cloud パートナー ポータルでテストおよび認定します。
9.  AppSource に公開して、ライブにします。

ここからは、これらの手順を詳細に説明します。
