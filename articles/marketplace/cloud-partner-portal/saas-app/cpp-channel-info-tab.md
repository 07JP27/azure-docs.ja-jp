# <a name="saas-application-channel-info-tab"></a>SaaS アプリケーションの [Channel Info]\(チャネル情報\) タブ

Go-To-Market (GTM) 資料や連絡先情報など、アプリのチャネル情報を提供するには、[Channel Info]\(チャネル情報\) タブを使用します。

![[Channel Info]\(チャネル情報\) フォーム](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>チャネル情報を構成するには

1. **[Channel Info]\(チャネル情報\) > [GTM materials]\(GTM 資料\)** で、GTM 資料をホストする Web サイトの URL を入力します。
2. **[Channel Manager Contact]\(チャネル マネージャー連絡先\)** で、サポートとビジネスの問題に関して Microsoft が利用できるマネージャーの連絡先を指定します。 次のフィールドの情報を入力します:名前、メール アドレス、電話番号。
3. **[保存]** を選択して、チャネル情報の設定を完了します。
