# <a name="lead-management-instructions-for-salesforce"></a>Salesforce での潜在顧客管理の手順

本書では、Salesforce システムをセットアップして Microsoft がセールス リードを提供できるようにする方法の手順を説明します。 

1.[セットアップ] に移動します。 <br/>
![Salesforce セットアップ イメージ](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

2. ナビゲーション バーで、[Build](ビルド)-> [Customize](カスタマイズ)->[Leads](潜在顧客)->[Web-to-Lead](Web-to-潜在顧客) の順に進みます。 <br/>
![Salesforce Web-to-Lead イメージ](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

3.[Create Web-to-Lead Form](Web-to-潜在顧客の作成) ボタンをクリックします。 <br/>
![Salesforce フォーム作成イメージ](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

4. 次のページでは、フォームのフィールドについて心配せずに、[Generate](生成) をクリックします。 <br/>
![Salesforce 生成イメージ](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

5. OID をコピーして、クラウド パートナー ポータルの<b>[Object Identifier]</b>(オブジェクト識別子) フィールドに貼り付けます。 <br/>
![Salesforce OID 取得イメージ](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
