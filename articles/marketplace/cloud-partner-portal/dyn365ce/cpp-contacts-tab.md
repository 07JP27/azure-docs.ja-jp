# <a name="azure-application-contacts-tab"></a>Azure アプリケーションの [連絡先] タブ

Dynamics 365 for Customer Engagement オファーに対するエンジニアリング向けおよび顧客向けのサポート情報を提供するには、 **[連絡先]** タブを使用します。

![オファーの [連絡先] タブ](./media/dynce-contacts-tab.png)


## <a name="specify-customer-and-engineering-contacts"></a>顧客およびエンジニアリングに対する連絡先を指定する

オファーの主要な**エンジニアリング連絡先**を定義するには、以下の値を指定します。 必須フィールドはアスタリスク (*) で示されます。

|    フィールド        |  説明                    |
|    ---------    |  ---------------                |
|  **[名前]\***       |  アプリのエンジニアリング関連の連絡先の名前です。 この連絡先が、Microsoft から技術的な通信を受け取ることになります。   |
|  **電子メール\***      |  エンジニアリング関連の連絡先のメール アドレスです  |
|  **電話\***      |  エンジニアリング関連の連絡先の電話番号です  |
|  |  |

オファーの主要な**サポート連絡先**を定義するには、以下の値を指定します。

|    フィールド        |   説明                   |
|    ---------    |  ---------------                |
|  **[名前]\***       |  アプリの顧客サポート連絡先の名前です。 この連絡先が、Microsoft からサポート関連の通信を受け取ることになります。   |
|  **電子メール\***      |  顧客サポートの連絡先のメール アドレスです  |
|  **電話\***      |  顧客サポートの連絡先の電話番号です  |
|  |  |
