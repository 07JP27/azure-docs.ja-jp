# クラウドと社内にまたがったハイブリッド アプリケーションの有効化

最新の IT 環境ではクラウド/内部設置型のハイブリッド アプリケーションが現実になっています。Service Bus により、既存の内部設置型アセットを利用して、信頼性の高い安全なソリューションを作成し、現在のインフラストラクチャを強化、拡張できます。
