# サンプル

[Power BI Embedded のパブリック プレビュー サンプルのダウンロード](http://go.microsoft.com/fwlink/?LinkID=761493)

出発点として、サンプル Web アプリが用意されています。このサンプルをダウンロードし、「[Get started with Microsoft Power BI Embedded sample (Microsoft Power BI Embedded のサンプルを使ってみる)](power-bi-embedded-get-started-sample.md)」をお読みください。このサンプルでは、次の方法を示しています。

  -	コンテンツをプロビジョニングする
      - ワークスペースを作成する
      - PBIX ファイルをインポートする
      - 接続文字列を更新し、データセットの資格情報を設定する

  - 安全にレポートを埋め込む
