# <a name="about-power-bi-embedded"></a>Power BI Embedded の概要

Power BI Embedded は、ISV や開発者が Power BI の機能を簡単に使えるようにするサービスです。 Power BI Embedded により、魅力的なビジュアル、レポート、ダッシュ ボードをアプリケーションにすばやく追加できるようになります。 Microsoft Azure 上にビルドされたアプリケーションと同様に、Machine Learning や IoT などのサービスを使用できます。 アプリケーションでのデータ探索を容易にすることで、ISV の顧客は情報に基づく的確な意思決定をすばやく行えるようになります。

> [!VIDEO https://www.youtube.com/embed/iEHfUuoZseo]

Microsoft は 2017 年 5 月に Power BI と Power BI Embedded のサービス統合を発表しました。 この統合により、1 つの API サーフェスで一貫性のある機能セットが提供されるようになり、両サービスにまたがって最新の機能を利用できるようになりました。 また、容量ベースの価格モデルが導入され、Power BI の使用が簡略化されました。

Power BI Embedded を使用すれば、ISV や開発者は Power BI API を使って、より柔軟にアプリケーションにインテリジェンスを埋め込めるようになります。 また、Microsoft の世界レベルの分析エンジンをアプリケーションに採用することで、開発に要する時間を最小化して市場投入までの時間を短縮し、差別化を図ることができます。 開発者は、ビジュアル分析機能を開発する代わりに、顧客のニーズを満たすソリューションの開発に時間を費やすことができます。 さらに Power BI Embedded では、Visual Studio や Azure などの使い慣れた開発環境を使用できます。

Power BI Premium を使って Power BI コンテンツを埋め込んだ既存のアプリケーションをお持ちの場合、 アプリケーションを提供する ISV や開発者も、それらを使用する組織も、特別な作業は必要ありません。 顧客を含めた誰もが、中断することなくアプリケーションを使い続けることができます。 Power BI ワークスペース コレクション上に構築された既存のアプリケーションがあり、統合 API サーフェスと新しい容量ベースの Azure SKU のご利用に関心をお持ちの方は、移行のガイダンスについてのドキュメントをご覧ください。

## <a name="comparing-power-bi-embedded-with-power-bi-premium"></a>Power BI Embedded と Power BI Premium の比較

Power BI Embedded は、アプリケーションを構築する開発者向けです。 アプリケーションのデータの視覚化を自分で構築する代わりに、サード パーティのビジネス インテリジェンスを使用する場合に役立ちます。 開発者は Power BI Embedded を使って、アプリケーションにダッシュボードとレポートを埋め込むことができます。 Power BI は、最も重要なビジネス データの単一ビューを組織に提供する、サービスとしてのソフトウェア (SaaS) 分析ソリューションです。

## <a name="easy-to-use-tools"></a>使いやすいツール

Power BI Embedded を使用すると、優れたアプリの構築に注力することができます。 既存のツールやスキルを使って、Power BI Embedded に対して管理や開発を行えます。

* [**Azure ポータル**](https://portal.azure.com/): すべての Azure サービスを管理するための Web ベースのアプリケーションです。
* [**Visual Studio Code**](https://code.visualstudio.com/docs): Windows、macOS、Linux 向けの無料でダウンロードできるオープンソースのコード エディターで、拡張機能に対応しています。
* [**Power BI Desktop**](https://powerbi.microsoft.com/desktop/): ビジュアル分析機能を備えたインタラクティブ レポートを作成できる無料でダウンロード可能なツールです。

Power BI Embedded では、REST API を使って任意の言語で開発できます。
