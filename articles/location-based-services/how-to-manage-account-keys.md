# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>Azure Location Based Services アカウントとキーを管理する方法

Azure Location Based Services アカウントとキーは、Azure Portal で管理することができます。 アカウントとキーを取得したら、Web サイトまたはモバイル アプリケーションで API を実装できます。

Azure サブスクリプションをお持ちでない場合は、開始する前に [無料アカウント](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) を作成してください。

## <a name="log-in-to-azure"></a>Azure にログインする 

Azure Portal (http://portal.azure.com) にログインします。

## <a name="create-a-new-account"></a>新しいアカウントを作成する

1. Azure Portal の左上にある **[新規]** ボタンをクリックします。

2. **Location Based Services** を検索して選択し、**[作成]** をクリックします。

3. Location Based Services アカウントの情報を入力します。 

![ポータルでアカウント情報を入力する](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>アカウント ページでキーを管理する

左側のメニューの [設定] で、**[キー]** をクリックします。

![ポータルでアカウント キーを管理する](./media/how-to-manage-account-keys/account-keys-portal.png)

キーを使用して Azure Location Based Services に対して認証し、位置情報ベースのアプリケーション開発を開始します。

## <a name="delete-an-account"></a>アカウントの削除

ナビゲーション ウィンドウの上部にある **[削除]** ボタンをクリックして、アカウントを削除できます。

![ポータルでアカウントを削除する](./media/how-to-manage-account-keys/account-delete-portal.png)
