# Azure ポータルにおける Azure Multi-Factor Authentication 設定の管理

Azure で多要素認証を使用する場合は、Azure ポータルで、追加のセキュリティ確認の設定を管理できます。

## Azure ポータルで追加のセキュリティ確認を設定する方法

<ol>
<li>Azure ポータルにログオンします。<li>Azure ポータルの上部に示されているユーザー名をクリックします。これにより、ドロップ ダウン ボックスが表示されます。<li>ドロップダウン ボックスで、[追加のセキュリティ確認] を選択します。[追加のセキュリティ確認] セクションが開きます。

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>確認ページが表示されます。設定を変更することもできます。</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/proofup.png)</center>
