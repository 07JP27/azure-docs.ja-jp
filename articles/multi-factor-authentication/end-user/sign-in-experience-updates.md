# <a name="updates-to-the-sign-in-experience-for-online-services"></a>オンライン サービスのサインイン エクスペリエンスの更新

現在、Office 365 や Azure などの Microsoft のオンライン サービス用のサインイン エクスペリエンスを更新しています。 これは視覚的な変更のみですが、画面が整理され、わかりやすくなるため、実行する必要がある操作 (つまり、サインイン) に集中しやすくなります。

## <a name="current-ui-versus-new-ui"></a>現在の UI と新しい UI

既存の UI と新しい UI を比較してみます。 新しい UI は、外観を更新したことで、より簡素化され、用途が絞られています。

![古い Microsoft サインイン エクスペリエンス](media/microsoft-sign-in-experience-old.jpg)

![新しい Microsoft サインイン エクスペリエンス](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>新しい UI はいつ表示されるようになるのでしょうか。

以下は、この変更がいつ予期されるかを示すタイムラインです。

| 期間 | Change |
|-----------|--------|
| 5 月初旬 | 既存のサインイン ページに通知バナーが表示されるようになります。 ここで変更内容を確認し、変更に備えることができます。 |
| 5 月中旬   | 通知バナーが変わり、更新された UI をプレビューするためのリンクが示されます。 プレビューでは、リンクを使用して既存のエクスペリエンスに戻すこともできます。 |
| 6 月中旬  | 更新されたサインイン UI を利用でき、すべてのユーザーに既定で新しい UI エクスペリエンスが表示されます。 |

## <a name="related-info"></a>関連情報

管理者の場合は、製品に基づき、これらの更新に関する詳細を参照できます。

- **Office 365**。 [Office 365 メッセージ センター](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail)で更新の詳細を確認できます。 このサイトにアクセスするには、管理者認証を使用してサインインする必要があります。

- **その他のオンライン サービス**。 こちらの[ブログ記事](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/)で更新の詳細を確認できます。
