# Office 365 ポータルにおける Azure Multi-Factor Authentication 設定の管理


Office 365 で多要素認証を使用する場合は、Office 365 ポータルで、追加のセキュリティ確認の設定を管理できます。

## Office 365 ポータルで追加のセキュリティ確認の設定を行う方法

<ol>

<li>[Office 365 ポータル](https://login.microsoftonline.com/) にログオンします。</li>
<li>上部にある小さな歯車のようなアイコンをクリックします。[Office 365 の設定] というドロップダウンが表示されます。</li>


<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365a.png)</center>

<li>[Office 365 の設定] をクリックします。設定ページが開きます。</li>
<li>左側の [追加のセキュリティ確認] をクリックします。</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365.png)</center>

<li>右側の [アカウントのセキュリティのために使用する電話番号を更新] というリンクをクリックします。新しいページにリダイレクトされます。電話またはモバイル アプリの使用法をもう一度確認する必要があります。</li>
<li>完了すると、確認ページが表示されます。設定を変更することもできます。</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-o365/addsec.png)</center>
