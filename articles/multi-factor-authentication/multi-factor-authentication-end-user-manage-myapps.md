# Myapps ポータルにおける Azure Multi-Factor Authentication 設定の管理


Office 365 で多要素認証を使用する場合は、Office 365 ポータルで、追加のセキュリティ確認の設定を管理できます。

## Myapps ポータルで追加のセキュリティ確認の設定を行う方法
<ol>
<li>[https://myapps.microsoft.com](https://myapps.microsoft.com) にサインインします。</li>
<li>上部にある [プロファイル] を選択します。</li>
<li>ユーザー名をクリックし、[追加のセキュリティ確認] を選択します。</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-myapps/myapp.png)</center>

<li>確認ページが表示されます。設定を変更することもできます。</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-myapps/proofup.png)</center>
