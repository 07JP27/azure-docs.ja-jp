# <a name="azure-app-service-security"></a>Azure App Service のセキュリティ
Azure App Service は、セキュリティ保護されたプラットフォームをお客様に提供するために、セキュリティと信頼性に関する主要な業界標準に準拠しています。 アプリの受信トラフィックと送信トラフィックの両方をセキュリティ保護するための使いやすい機能が複数用意されています。 また、お客様のアプリケーション コードをセキュリティ保護するために、Web アプリケーションの脆弱性をスキャンする、外部から提供された機能が用意されています。

[!INCLUDE [app-service-blueprint-security](../../includes/app-service-blueprint-security.md)]

Azure App Service では、Azure の Cloud Services および Virtual Machines で使用されるのと同じマルウェア対策ソリューションを使用します。 詳細については、 [マルウェア対策についてのドキュメント](../security/azure-security-antimalware.md)
