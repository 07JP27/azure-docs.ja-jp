# <a name="azure-government-marketplace"></a>Azure Government Marketplace
Azure Marketplace for Government へのサービスの発行に興味をお持ちのパートナー様は、以下の詳細をご覧ください。

## <a name="publishing"></a>発行
> [!NOTE]
> Azure 認定 Marketplace パートナー以外のお客様は、先に進む前に[こちら](../marketplace-publishing/marketplace-publishing-getting-started.md)の手順を完了してください。
> 
> 

### <a name="step-1"></a>手順 1
[https://publish.windowsazure.com](https://publish.windowsazure.com) にログオンします。

### <a name="step-2"></a>手順 2.
発行するサービスをクリックします。

### <a name="step-3"></a>手順 3.
**[SKUS (SKU)]** をクリックし、[Azure Government Cloud (Azure Government クラウド)] ボックスをオンにします。

> [!NOTE]
> サポートされているのはライセンス持ち込み (BYOL) SKU のみです。  このオプションは、従量課金の (PayG) SKU では利用できません。
> 
> 

![alt text](./media/government-manage-marketplace-partner-1.png)

### <a name="step-4"></a>手順 4.
[+ Add Certification (証明書の追加)] リンクをクリックし、発行するサービスの証明書へのリンクを追加します。

![alt text](./media/government-manage-marketplace-partner-2.png)

### <a name="step-5"></a>手順 5.
Microsoft Azure Government クラウドの試用版アカウントをリクエストし、発行ポータル ([https://azuregov.microsoft.com/trial/azuregovtrial](https://azuregov.microsoft.com/trial/azuregovtrial)) でイメージをテストできるようにします。

米国連邦政府、州政府、地方自治体、または部族の各組織にサービスを提供するパートナーとしてのお客様の資格は、検証が行われたうえで、電子メール経由で承認されます。  試用版アカウントは 3 ～ 5 営業日以内に利用可能になります。

### <a name="step-6"></a>手順 6.
[発行]、[Push to Staging (ステージングにプッシュ)] の順にクリックします。 

![alt text](./media/government-manage-marketplace-partner-3.png)

ステージングされたサービスにアクセス可能な、ホワイトリストに登録済みのサブスクリプションを入力するよう求めるメッセージが表示されます。 新たに取得した Azure Government 試用版アカウントのサブスクリプション ID を入力します。

![alt text](./media/government-manage-marketplace-partner-4.png)

### <a name="step-7"></a>手順 7.
サービスのステージングが成功したら、Azure Government 試用版アカウントを使用して [https://portal.azure.us](https://portal.azure.us) にログインすることで、イメージをテストできます。

### <a name="step-8"></a>手順 8.
試用版サブスクリプションを使用してイメージを検証したら、[発行] をクリックして運用環境への移行の承認依頼を行うことで、サービスの提供を開始できます。 

![alt text](./media/government-manage-marketplace-partner-5.png)

## <a name="next-steps"></a>次のステップ
