# Windows Server が実行されている仮想マシンにログオンする方法#

Windows Server を実行する仮想マシンにログオンするには、管理ポータルの [接続] ボタンを使用して、リモート デスクトップ接続を開始します。(Linux VM の場合、[実行されている Linux の仮想マシンにログインする方法](virtual-machines-linux-how-to-log-on) をご覧ください)
