# Azure での Linux 仮想マシンでホストされている Oracle ソフトウェア

[AZURE.INCLUDE [virtual-machines-common-oracle-support](../../includes/virtual-machines-common-oracle-support.md)]

Azure インフラストラクチャ サービス内の Linux 仮想マシン上で実行するために、1 番目または 2 番目の開発およびテスト、または運用 Oracle データベースを設定します。構成のしやすさ、また Oracle データベースのワークロードをすばやく拡張して新しい容量を含める機能をご活用ください。
