# Azure での Windows 仮想マシンでホストされている Oracle ソフトウェア


Azure インフラストラクチャ サービス内の Windows 仮想マシン上で実行するために、1 番目または 2 番目の開発およびテスト、または運用 Oracle データベースを設定します。構成のしやすさ、また Oracle データベースのワークロードをすばやく拡張して新しい容量を含める機能をご活用ください。[Linux 仮想マシン](virtual-machines-linux-classic-oracle.md)上で Oracle ソフトウェアを設定することもできます。
