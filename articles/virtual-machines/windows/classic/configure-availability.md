# <a name="how-to-configure-an-availability-set-for-windows-virtual-machines-in-the-classic-deployment-model"></a>クラシック デプロイメント モデルに Windows 仮想マシンの可用性セットを構成する方法
> [!IMPORTANT] 
> Azure には、リソースの作成と操作に関して、 [Resource Manager とクラシック](../../../resource-manager-deployment-model.md)の 2 種類のデプロイメント モデルがあります。 この記事では、クラシック デプロイ モデルの使用方法について説明します。 最新のデプロイでは、リソース マネージャー モデルを使用することをお勧めします。 Resource Manager デプロイメントで[可用性セットを構成する](../create-availability-set.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json)こともできます。
