# <a name="workloads-on-azure-virtual-machines"></a>Azure Virtual Machines 上のワークロード

Azure Virtual Machines では、多くのワークロードをサポートできます。 各ワークロードには、Azure VM での展開に最適な設計に関する特殊なガイダンスを利用できます。 特殊なワークロード向けに提供できるガイダンスの一部を次に示します。

- Azure の[ハイ パフォーマンス コンピューティング](https://docs.microsoft.com/azure/architecture/topics/high-performance-computing/)では、CPU または GPU に最適化された仮想マシンを使用して、複雑なコンピューティング集中型の問題を解決します。 
- Azure の [SAP](./sap/get-started.md) はミッション クリティカルな SAP ワークロードとシナリオを、拡張性が高く、エンタープライズで実証済みのプラットフォームで実行できます。
- [Oracle](./oracle/oracle-considerations.md) では、Oracle Linux に基づく仮想マシン イメージで、Azure における Oracle DB 12.1 Standard Edition および Enterprise Edition の実行をサポートしています。 
- [メインフレームのホスト変更](./mainframe-rehosting/overview.md)
