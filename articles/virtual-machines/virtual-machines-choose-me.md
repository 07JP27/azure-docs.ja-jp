# Azure の仮想マシンを選択する必要がありますか。

[AZURE.INCLUDE [learn-about-deployment-models](../../includes/learn-about-deployment-models-include.md)]

Azure にはアプリケーションを実行するための複数のホスティング モデルがあります。それぞれの実行モデルには異なるサービスが用意されているため、実行する内容に正確に基づいて実行モデルを選択してください。Azure のVM はあなたにとっての選択肢ですか。
