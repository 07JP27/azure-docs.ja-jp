# <a name="azure-remoteapp-retirement"></a>Azure RemoteApp の提供終了
この記事は、Microsoft から連絡が取れず、Azure RemoteApp の提供終了について了解が得られていないお客様を対象に表示しています。 

Azure RemoteApp の提供が 2017 年 8 月 31 日をもって終了することをお客様が把握していることを確認するために、Microsoft はお客様をはじめとする Azure サブスクリプション管理者への連絡を試みております。[Azure RemoteApp の提供終了については、2016 年 8 月 12 日に告知](http://aka.ms/araretirement)いたしました。   恐れ入りますが、次の点について至急メール ([arainfo@microsoft.com](mailto:arainfo@microsoft.com)) でお知らせいただきますようお願い申し上げます。
1.  2017 年 8 月 31 日の提供終了をお客様および所属する組織が把握していること。
2.  お客様の移行プランをお聞かせください。必要であればお手伝いいたします。 移行の選択肢については、「[Azure RemoteApp から移行する際の選択肢](http://aka.ms/aramigration)」に詳しい説明があります (Citrix XenApp Essentials、MyCloudIT、Frame など)。 

Azure 管理ポータル内の REMOTEAPP ノードはブロックされている可能性があります。Microsoft から連絡が取れないために、強制的にお客様からの連絡をお待ちする形を取る必要があったためです。  Azure サブスクリプションに関連付けられているメール アドレスをだれもチェックしていないか、またはメール アドレスが無効であることが原因で、Microsoft からのメールをまだご覧になっていない可能性があります。  Azure サブスクリプションに関連付けられているメール アドレスが有効であり、チェックされているかどうかを確認するよう強くお勧めします。Azure Portal にアクセスし、サブスクリプション管理者をご確認ください。  

ご不明な点がある場合や移行サポートをご希望の場合は、  メール ([arainfo@microsoft.com](mailto:arainfo@microsoft.com)) でご連絡いただくか、ここで移行チームとの[ミーティングを予約](http://aka.ms/ericorman)してください。
