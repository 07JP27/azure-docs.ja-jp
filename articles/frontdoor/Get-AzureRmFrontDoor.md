# Get-AzureRmFrontDoor
Front Door の構成を取得します。

## 構文

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Description
**Get-AzureRmFrontDoor** cmdletGet により、指定した情報に一致する現在のサブスクリプションの Front Door がすべて取得されます。
