# Azure SQL Database 弾力性データベース ツールのトピック

Azure SQL DB の弾力性データベース機能はデータ階層の開発と管理を簡単にするために設計されています。大量のデータベースを利用して動的エンドユーザー基盤をサポートする Software as a Service (SaaS) の開発者を特に念頭に置いています。

**弾力性データベース ツール**を利用すると、シャーディング パターンを活用してさまざまなデータベースにまたいでスケールアウトするアプリケーションを構築し、管理できます。たとえば、シャード管理、データ依存ルーティング、ADO.Net のシャード間クエリ機能をサポートする弾力性データベース クライアント ライブラリが含まれています。また、シャード間のデータ移動を可能にしてシャード管理を簡易化する分割/マージ ツールも含まれています。このツールは弾力性データベース プールを補い、予測可能な価格設定を可能にします。データベースの数に関係なくパフォーマンス特性は保証されます。関連するジョブ機能は、T-SQL スクリプトでプール内のあらゆるデータベースをアップグレードし、管理するための簡単で信頼できるツールを提供します。
