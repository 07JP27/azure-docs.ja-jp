# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Azure SQL Database サーバーとその設定の表示および更新

Azure SQL Database サーバーとその設定を表示および更新するには、[Azure Portal](sql-database-manage-portal.md)、[[PowerShell]](sql-database-manage-powershell.md)、REST API、または C# を使用します。 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Azure Portal を使用して Azure SQL サーバー設定を表示および更新
1. [Azure Portal](https://portal.azure.com/) で **[SQL Server]** ブレードを開きます。 

    ![新しい SQL Server](./media/sql-database-get-started/new-sql-server.png)

2. 対象のサーバーをクリックし、[SQL サーバー] ブレードで目的の設定をクリックします。 

    ![[SQL Server] ブレード](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Azure Portal の入門用チュートリアルについては、「[Azure Portal と SQL Server Management Studio を使用して Azure SQL Database のサーバー、データベース、ファイアウォール規則を使ってみる](sql-database-get-started.md)」を参照してください。
>
