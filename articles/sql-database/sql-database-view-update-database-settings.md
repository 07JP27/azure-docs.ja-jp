# <a name="view-and-update-an-azure-sql-database-and-its-settings"></a>Azure SQL データベースとその設定の表示および更新

Azure SQL データベースとその設定を表示および更新するには、[[Azure Portal]](sql-database-manage-portal.md)、[[PowerShell]](sql-database-manage-powershell.md)、[[SQL Server Management Studio]](sql-database-manage-azure-ssms.md)、REST API、または C# を使用します。 

## <a name="view-and-update-sql-database-settings-using-the-azure-portal"></a>Azure Portal を使用して SQL データベース設定を表示および更新

1. [Azure Portal](https://portal.azure.com/) で **[SQL データベース]** ブレードを開きます。 

    ![SQL データベース](./media/sql-database-get-started/sql-databases.png)

2. 対象のデータベースをクリックし、[SQL データベース] ブレードで目的の設定をクリックします。

    ![新しいサンプル データベースのブレード](./media/sql-database-get-started/new-sample-db-blade.png)

> [!TIP]
> Azure Portal の入門用チュートリアルについては、「[Azure Portal と SQL Server Management Studio を使用して Azure SQL Database のサーバー、データベース、ファイアウォール規則を使ってみる](sql-database-get-started.md)」を参照してください。
>
