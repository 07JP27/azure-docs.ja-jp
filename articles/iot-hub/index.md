#<a name="iot-hub-documentation"></a>IoT Hub のドキュメント

Azure IoT Hub は、何百万もの IoT デバイスとソリューション バックエンド間で、セキュリティで保護された信頼性のある双方向通信を実現する、完全に管理されたサービスです。  また、デバイス接続の詳細な監視を実現し、よく使われる言語とプラットフォーム向けのデバイス ライブラリが含まれています。
