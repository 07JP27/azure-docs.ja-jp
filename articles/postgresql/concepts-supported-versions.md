# <a name="supported-postgresql-database-versions"></a>サポートされている PostgreSQL Database バージョン
Microsoft は、Azure Database for PostgreSQL サービスで、n-2 バージョン (つまり、現在リリースされているメジャー バージョン (n) と以前の 2 つのメジャー バージョン (-2)) の PostgreSQL エンジンをサポートすることを目指しています。

Azure Database for PostgreSQL では現在、次のバージョンがサポートされています。

## <a name="postgresql-version-103"></a>PostgreSQL のバージョン 10.3
PostgreSQL 10.3 の機能強化と修正の詳細については、[PostgreSQL のドキュメント](https://www.postgresql.org/docs/10/static/release-10-3.html)を参照してください。

## <a name="postgresql-version-967"></a>PostgreSQL のバージョン 9.6.7
PostgreSQL 9.6.7 の機能強化と修正の詳細については、[PostgreSQL のドキュメント](https://www.postgresql.org/docs/9.6/static/release-9-6-7.html)を参照してください。

## <a name="postgresql-version-9511"></a>PostgreSQL のバージョン 9.5.11
PostgreSQL 9.5.11 の機能強化と修正については、[PostgreSQL のドキュメント](https://www.postgresql.org/docs/9.5/static/release-9-5-11.html)を参照してください。

## <a name="managing-updates-and-upgrades"></a>更新プログラムとアップグレードの管理
Azure Database for PostgreSQL では、マイナー バージョンの更新プログラムの適用については自動管理されます。 現在、メジャー バージョンのアップグレードはサポートされていません。 たとえば、PostgreSQL 9.5 から PostgreSQL 9.6 へのアップグレードはサポートされません。

## <a name="next-steps"></a>次の手順
