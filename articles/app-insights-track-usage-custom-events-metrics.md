# カスタム イベントおよびメトリックを使用した利用状況の追跡

### 近日対応予定

Application Insights SDK を使用すると、ユーザーのアクティビティを監視するためのコードをアプリケーションに挿入できるため、ユーザーのニーズを満たすようにアプリケーションを調整することが可能になります。

現在、この機能を Microsoft Azure の新しい Application Insights ポータルに移植する作業の途中にあります (必要に応じて、[古いポータルでの動作を確認][古いポータルでの動作を確認]できます)。

現時点では、[Trace、NLog、または Log4Net で生成された診断イベント ログをキャプチャして検索][Trace、NLog、または Log4Net で生成された診断イベント ログをキャプチャして検索]することができます。
