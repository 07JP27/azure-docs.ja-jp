#Azure Mobile Engagement Web SDK のリリース ノート

##2\.0.1 (06/10/2016)

-   IE8 と IE9 で SDK を無効にしました。
-   Opera Web ブラウザーの検出を修正しました。

##2\.0.0 (06/07/2016)

-   Azure Mobile Engagement の最初のリリース。
-   appId 構成が接続文字列構成に置き換わりました。
-   エージェントを有効/無効にする API を追加しました。
-   セキュリティの強化。
-   ブラウザーのネイティブ JSON API が使用されます。
-   デバイス間でメッセージを送受信する API が削除されました。
-   推奨されない XMPP 接続に関連したコールバックを削除しました。 
-   このバージョンでは、Reach の機能がサポートされません。
