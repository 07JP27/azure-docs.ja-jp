# <a name="mobile-engagement-documentation"></a>Mobile Engagement のドキュメント

Azure Mobile Engagement は、サービスとしてのソフトウェア (SaaS) 型のユーザー エンゲージメント プラットフォームで、これによりデータ主導のアプリ使用状況分析、ユーザーのセグメント化、プッシュ通知とアプリ内メッセージングが可能になります。
