# David Chappell による DocumentDB に関するホワイトペーパー 

**概要:** Microsoft Azure の管理された NoSQL データベースである DocumentDB について説明します。DocumentDB は、次の場合にお勧めのデータベース テクノロジです。

* ご使用のアプリケーションがスキーマ不要のデータ モデルの利点を活用する場合。
* データベースを多数のストレージおよびスループットに合わせて拡張する必要がある場合。
* よく把握しているクエリ言語および開発プラットフォームを使用する必要がある場合。

また、このホワイトペーパーでは、以下の基礎的な概念も取り上げます。

* DocumentDB データ モデル
* アプリケーションがデータを処理する方法
* パフォーマンスと整合性のバランスを取るためにアプリケーションに備わっているオプション
