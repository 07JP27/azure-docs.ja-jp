#Azure サポートの購入方法は?
Azure では、クラウドでアプリケーションの開発を開始する開発者からビジネス クリティカルなアプリケーションをデプロイする企業まで、あらゆるお客様向けの柔軟なサポート オプションをご用意しています。課金とサブスクリプションの管理に関する問題に対するサポートは無償で提供されますが、Azure のテクニカル サポートについては Azure サポートのサブスクリプションを購入する必要があります。

購入できる Azure のサブスクリプションについては、[ここ](https://azure.microsoft.com/support/legal/offer-details/)をクリックしてください。テクニカル サポートについては、「[Azure のサポート プラン](https://azure.microsoft.com/support/plans/)」ページを参照してください。

セルフヘルプ リソースも利用できます。Azure サポートに関する FAQ を[ここ](https://azure.microsoft.com/support/faq/)で確認できます。また、すべての Azure.com ページの [ドキュメント] タブからサポート記事にアクセスできます。
