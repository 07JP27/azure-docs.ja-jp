# リリース ノート

## 2.0.0 (02/17/2015)

-   Azure モバイル エンゲージメントの最初のリリース
-   appId/sdkKey 構成が接続文字列の構成で置き換えられました。
-   任意の XMPP エンティティから任意の XMPP メッセージを送受信する API が削除されました。
-   デバイス間でメッセージを送受信する API が削除されました。
-   セキュリティが強化されました。
-   SmartAd の追跡が削除されました。
