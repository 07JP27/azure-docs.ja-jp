# 新しい Mobile Services を作成する方法

このトピックでは、[Azure 管理ポータル][Management Portal]を使用して、サーバー側ビジネス ロジックに JavaScript を使用する新しい JavaScript バックエンドを Azure Mobile Services で作成する方法について説明します。モバイル サービス バックエンドと、新しいモバイル サービスを使用するモバイル クライアント アプリケーションの両方を作成するには、代わりに「[Mobile Services の使用]」をご覧ください。

>[AZURE.NOTE]このチュートリアルを完了するには、Azure Mobile Services の機能を有効にした Azure アカウントが必要です。アカウントがない場合は、無料の試用アカウントを数分で作成できます。詳細については、「 <a href="http://www.windowsazure.com/pricing/free-trial/?WT.mc_id=A30A4DDE2&amp;returnurl=http%3A%2F%2FFja-jp%2Fdocumentation%2Farticles%2Fmobile-services-how-to-create-new-service%2F" target="_blank">Azure 無料評価版</a>」をご覧ください。
