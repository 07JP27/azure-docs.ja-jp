# <a name="commands"></a>コマンド
|||
|---|---|
|az mysql server|サーバーを管理します。 サーバーには、データベースやデータ ウェアハウス、エラスティック プールが存在します。|
|az mysql server create|MySQL サーバーを作成します。|
|az mysql server delete|MySQL サーバーを削除します。|
|az mysql server list|リソース グループに存在する一連の MySQL サーバーをすべて取得します。|
|az mysql server show|MySQL サーバーの詳細を表示します。|
|az mysql server update|サーバーを更新します。|
|az mysql server restore|サーバーを以前の状態に復元します。|
|az mysql server firewall-rule|サーバーのファイアウォール規則を管理します。|
|az mysql server firewall-rule create|ファイアウォール規則を作成します。|
|az mysql server firewall-rule delete|ファイアウォール規則を作成します。|
|az mysql server firewall-rule list|ファイアウォール規則の一覧を表示します。|
|az mysql server firewall-rule show|ファイアウォール規則の詳細を表示します。|
|az mysql server firewall-rule update|ファイアウォール規則を更新します。|
|az mysql db|データベースを管理します。|
|az mysql db list|サーバーにあるすべての MySQL データベースを一覧表示します。|
|az mysql server-logs list|サーバーのログ ファイルを一覧表示します。|
|az mysql server-logs download|サーバーからローカルにログをダウンロードします。|
|az mysql server configuration list|サーバー構成パラメーターを一覧表示します。|
|az mysql server configuration show|サーバー構成パラメーターの詳細を表示します。|
|az mysql server configuration update|サーバー構成パラメーターの値を変更します。|
|||
