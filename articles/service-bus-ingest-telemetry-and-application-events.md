# 大規模な利用統計情報とアプリケーション イベントの取り込み
 
最新のアプリケーションは、実行中に厖大な量の情報を作成します。それがモバイル アプリケーションのユーザーのアクションでも Web サイトのクリックストリームでも、この利用統計情報は、長期的な分析、リアルタイムの稼動状況の監視や意思決定支援に役立ちます。Service Bus はこの情報を取得し、処理するのに必要なツールを提供します。
