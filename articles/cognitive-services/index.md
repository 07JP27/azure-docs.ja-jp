#<a name="cognitive-services-documentation"></a>Cognitive Services のドキュメント

Cognitive Services (プレビュー) は、アプリケーション内でコンテキストに応じた自然な操作を実現できる、一連の API です。  Cognitive Services は、言語、音声、視覚、検索、知識の API を提供します。
