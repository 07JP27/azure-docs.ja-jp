# <a name="what-is-the-linguistic-analysis-api-preview"></a>Linguistic Analysis API とは (プレビュー)

> [!IMPORTANT]
> Linguistic Analysis プレビューは、2018 年 8 月 9 日に使用停止になりました。 テキスト処理と分析には、[Azure Machine Learning テキスト解析モジュール](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics)の使用をお勧めします。

Linguistic Analysis API で、テキストの構造を識別する自然言語処理 (NLP) ツールにアクセスすることができます。 現在のリリースには、次の 3 つのタイプの分析が用意されています。

- [文の分割とトークン化](Sentences-and-Tokens.md)
- [品詞のタグ付け](POS-tagging.md)
- [構文解析](Constituency-Parsing.md)

## <a name="getting-started"></a>Getting Started (概要)

上記の主要なタイプの分析に慣れたら、[使用可能なアナライザーのリストを取得](AnalyzersMethod.md)する際のエンドポイントとそれらに[名前を付ける](Analyzer-Names.md)方法、および[新しい入力を分析](AnalyzeMethod.md)する際のエンドポイントについて調べます。
