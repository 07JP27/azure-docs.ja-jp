# <a name="speech-sdk-reference"></a>Speech SDK リファレンス

Speech ソフトウェア開発キット (SDK) は、音声認識対応のソフトウェア開発を支援します。アプリケーションから SDK を利用することで、Speech サービスの各種機能を利用することができます。 現在、この SDK を通じて、**音声テキスト変換**と**音声翻訳**をご利用いただけます。 テキスト読み上げには、HTTP 経由の [REST] (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) POST 呼び出しが使用されます。

以下の表は、現在サポートされているプログラミング言語とオペレーティング システムを表したものです。

|プログラミング言語|プラットフォーム|API リファレンス|
|-|-|-|
|C#|Windows|[Browse](https://aka.ms/csspeech/csharpref)
|C/C++|Windows、Linux|[Browse](https://aka.ms/csspeech/cppref)
|Java\*|デバイス|[Browse](https://aka.ms/csspeech/javaref)

\* *Java SDK は、[Speech Devices SDK](speech-devices-sdk.md) の一部です。*
