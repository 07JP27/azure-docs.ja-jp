# <a name="text-moderation---supported-languages-by-feature"></a>テキストのモデレート - 機能別のサポートされている言語

> [!NOTE]
> **language** パラメーターについては、`eng` を割り当てるか、パラメーターを空のままにして、コンピューター支援による**分類**応答を表示します (プレビュー機能)。 **この機能でサポートされているのは英語のみです**。
