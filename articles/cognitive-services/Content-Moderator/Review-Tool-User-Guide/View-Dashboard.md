# <a name="view-dashboard"></a>ダッシュボードの表示

**[ダッシュボード]** タブをクリックして、ツール内で実行されたすべてのコンテンツ レビューの主要なメトリックを表示します。 画像とテキストの合計、完了、および実行中のレビューを表示できます。 既定のチームとサブチームのメトリックを表示できます。
